介紹

使用 Brightcove Beacon，您將收到一個生成的設備應用程序，該應用程序可以添加到松下商店。

  • 店鋪詳情： https://developer.vieraconnect.com/
  • 建議審批時間：3週

Panasonic的必填信息：

  • 應用
    • 應用程序URL和應用程序登台URL
  • 描述
    • 最多1024個字符
    • 最多26條線
    • 每行最多包含85個字符（或44個亞洲字符）
    • 請在適當的地方插入換行符
  • 圖示
    • 需要四個圖標圖像：
      • 160x112（PNG）
      • 橫幅樣式圖標256x60（PNG）
      • 360x252（PNG）
      • 圓形圖標直徑 336x336 (PNG)
  • 截屏
    • 至少需要一個
    • 480w x 270h PNG
    • 它們顯示在市場的應用程序詳細信息頁面上

入門

要訪問 Panasonic 開發人員頁面，請按照下列步驟操作：

  1. 在瀏覽器中，導航到松下開發者登錄頁。

  2. 使用您的帳戶登錄或創建一個新帳戶。

  3. 登錄後，您應該會看到控制面板。

  4. 閱讀位於右側的註釋提交新申請按鈕。選擇主頁關聯。

  5. 查看應用程序要求。不用擔心，Brightcove將使用所需的技術來創建應用程序。

申請信息

這申請資料部分是您輸入基本應用程序信息的地方。

  1. 在控制面板中，選擇提交新申請按鈕。

  2. 您應該會看到這樣的輸入表單：

  3. 在裡面申請信息頁面上，輸入必填字段的值：

    • 項目類型 -從下拉菜單中選擇可用的最新HTML版本。
    • 申請網址 -創建兩個存儲庫以託管來自Brightcove的應用程序構建：一台用於生產，一台用於分期。應用程序 URL 是生產 URL。
    • 暫存網址 - 輸入應用程序構建的暫存 URL。
    • 版本號 - 輸入您的應用程序版本。

  4. 選擇添加/刪除按鈕。選擇您想要用於您的應用的語言。選擇確認按鈕。

  5. 輸入具有所選語言的字段的值。在這個例子中英語（北美）被選中。

    • 預設語言 -從下拉菜單中，為您的應用選擇初始語言。如果您的應用程序支持英語，那麼Panasonic建議您選擇英語（（北美）或英語（英國型號））。
    • 應用名稱 -輸入您要用於應用程序的名稱。
    • 簡稱 - 輸入您要用於您的應用程序的短名稱。這可以與應用程序名稱相同。
    • 關鍵字詞 -輸入要由市場搜索功能使用的關鍵字來找到您的應用。用逗號分隔關鍵字（例如：乒乓球、體育）。最多128個字符。
    • 描述 -輸入您的應用說明。這將顯示在市場的應用程序詳細信息頁面上。在輸入框中單擊以獲取詳細信息。
    • 終止用戶許可協議 - 可選的。輸入最終用戶許可協議。

  6. 在所選語言列表中選擇下一種語言，並為上一步中描述的字段輸入值。

  7. 選擇下一個按鈕。

額外細節

這額外細節部分允許您添加支持和內容詳細信息。

  1. 選擇後下一個上一節中的按鈕，您應該會看到類似以下的輸入表單：

  2. 在裡面額外細節頁面上，輸入必填字段的值：

    • 支持郵箱 - 提供電子郵件地址，供用戶在遇到您的應用程序問題時聯繫。
    • 家長評分 -展開下拉菜單，查看每個家長評分的定義。選擇適合您的應用的評級。
    • 巴西內容分級 -如果您打算在巴西發行，這是必需的。展開下拉菜單，以查看每個巴西等級的定義。選擇合適的評級。
    • 聯繫電子郵件 -輸入您的聯繫電子郵件地址。
    • 類別 -展開下拉菜單以查看類別列表。選擇合適的類別。
    • 價格 -松下上的應用程序通常可以免費下載。
    • 在應用程序內購買 -選擇您的應用是否將進行應用內購買。
    • 市場地點 -點擊選擇市場按鈕以查看國家/地區列表。要么選擇所有國家或選擇您希望應用可用的國家/地區。完成後，選擇完畢按鈕。
    • 質量檢查說明 -包括所有可能有用的說明，包括您添加了按鈕，更改了鏈接等。如有必要，請為您的應用添加測試帳戶。

  3. 選擇下一個按鈕。

上傳資產

這上傳資產部分允許您添加圖標和圖像。

  1. 選擇後下一個上一節中的按鈕，您應該會看到類似以下的輸入表單：

  2. 在裡面上載資產頁面上，輸入必填字段的值：

    • 應用說明文檔 -創建一個詳細說明您的應用程序工作方式的文檔。這可以是您為 LG 創建的同一文檔。選擇瀏覽按鈕來定位您的文件。然後，選擇上傳按鈕。

      注意只有doc，docx或者pdf允許上傳文件。檔案大小上限為8MB。

    • 圖標1 -上傳市場圖標，該圖標將顯示在列表頁面和應用程序詳細信息頁面在市場上。

      160w x 112h PNG

    • 圖標2（圓形） -上傳Firefox OS圖標，該圖標將顯示在Apps Doc頁面在市場上。

      336像素直徑圓形PNG

    • 示例圖像 1 - 這些是您的應用程序的屏幕截圖，將顯示在申請詳情頁面在市場上。最多添加5個示例圖像。

      480w x 270h PNG

  3. 選擇下一個按鈕。

同意條款和條件

這同意條款和條件部分允許您在提交到商店之前查看您為應用程序添加的所有信息。還有一個部分包含松下條款和條件。

  1. 選擇後下一個上一節中的按鈕，您應該會看到類似以下的輸入表單：

  2. 查看您為應用添加的信息和圖像。

  3. 閱讀Panasonic電視的條款和條件。選中表明您已閱讀並同意的框。

  4. 選擇提交按鈕。

  5. 在應用清單頁面上，您應該會看到您的應用已另存為草稿。

  6. 您的應用將在上線前進入 QA 狀態。
