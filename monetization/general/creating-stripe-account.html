概述

需要Stripe帳戶以允許來自客戶的以下平台的付款：

  • 網絡
  • 三星智能電視
  • 索尼智能電視
  • 東芝智能電視

這意味著 Web 應用程序和所有 Smart TV 應用程序都需要 Stripe 以外 安卓電視和消防電視。

以下步驟概述了將 Stripe 與 Brightcove Beacon 結合使用的過程：

  1. 通過提供基本信息（電子郵件，名稱，密碼）來創建您的帳戶。
  2. 通過提供有關您的業務的信息來激活您的帳戶。
  3. 檢索Stripe API密鑰。
  4. 向 Brightcove 人員提供 API 密鑰，以便將他們添加到 Brightcove Beacon Experience。

創建和激活Stripe帳戶

  1. 瀏覽到條紋首頁。
  2. 點擊現在開始按鈕。
  3. 填寫您的基本信息以創建帳戶。
  4. 檢查以確保已顯示Stripe儀表板。
  5. 點擊激活您的Stripe帳戶選項。
  6. 點擊現在開始按鈕開始該過程。
  7. 您需要驗證您的電子郵件地址。
  8. 驗證電子郵件地址後，您會看到要填寫的表格以激活您的帳戶。表格的主要部分以黃色突出顯示。
  9. 填寫完表格後，點擊遞交申請按鈕。

獲取您的Stripe API密鑰

您的 Stripe 帳戶和 Brightcove Beacon 之間的連接是通過 Stripe API 密鑰完成的。在上面顯示的儀表板的屏幕截圖中，您會看到一個指向獲取您的實時API密鑰。請注意，在激活帳戶之前，您只能獲得測試API密鑰。

在資訊主頁中，按一下獲取您的實時API密鑰並按照說明進行下載。

將條紋連接到Brightcove信標

擁有Stripe API密鑰後，會將其提供給配置Brightcove信標體驗的Brightcove人員。他們會將您的 Stripe API 密鑰與您的 Brightcove Beacon Experience 相關聯，以便在您配置要在您的應用程序中使用的 Stripe 時，它會與您的帳戶相關聯。
