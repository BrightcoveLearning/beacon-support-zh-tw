介紹

Evergent CRM 與 Brightcove Beacon 集成，允許您管理查看者帳戶、訂閱和交易。本指南涵蓋了 CRM 的使用。

登錄賬戶管理

  1. 打開網絡瀏覽器。
  2. 在地址欄中，輸入Evergent提供的應用程序URL。
  3. 輸入有效的電子郵件/用戶名和密碼。
  4. 點擊登入按鈕開始您的 Evergent 會話。
  5. 有權訪問多個業務單位的用戶必須從可用的下拉列表中選擇業務單位，然後單擊提交：
  6. 點擊帳戶管理儀表板主屏幕上的圖像按鈕以進入“帳戶管理"應用程序：
  7. 彈出的模式對話框將要求您從以下選項中進行選擇：
    • 創建帳號
    • 現有帳戶
    • 訂單不完整
  8. 選擇現有賬戶

帳戶管理

賬戶管理模塊用於維護客戶詳細信息、服務、訂單、賬單和財務等，以建立與 Evergent 現有和潛在客戶的關係。

管理現有帳戶

進入“現有帳戶"部分後，您可以搜索要使用的帳戶。

  1. 在帳戶搜尋屏幕，在搜索任何內容部分中，輸入與帳戶相關的詳細信息，以通過以下方式搜索現有帳戶帳戶類型，業務部， 或者賬號：
  2. 要查看更多搜索選項，請啟用高級搜索字段撥動開關：
  3. 高級搜索選項允許您使用多個搜索字段的任意組合來搜索帳戶：
    進階搜尋欄位
    場地 描述
    選擇業務單位 搜索特定業務部門下的客戶
    賬號 在 Evergent 系統中成功註冊時生成的客戶唯一帳號
    CP客戶編號 客戶購買的硬件訂閱的唯一序列號。
    註冊號碼 客戶在 Evergent 系統中成功註冊時生成的唯一註冊號
    SP 帳戶 ID 供應系統中客戶的唯一 SP 帳戶 ID
    訂單編號 成功購買訂閱後為客戶生成的唯一訂單 ID。
    發票號碼 成功交付賬單後為客戶生成的唯一發票 ID
    CTN 唯一客戶電話號碼
    外部訂單編號 成功購買外部訂單後為客戶生成的唯一外部訂單 ID
    付款資料編號 支付時生成的唯一支付ID
    禁止 與客戶帳戶相關的唯一 BAN
    用戶名 根據最新客戶記錄的客戶登錄用戶名
    名 根據最新客戶記錄的客戶名字
    姓 根據最新客戶記錄的客戶姓氏
    社會保障號碼 註冊期間根據客戶的最新記錄提供的客戶的SSN（社會保險號）
    備用用戶名 註冊期間提供的客戶的備用用戶名。此字段在保存期間保存任何電話號碼或電子郵件或字母數字值
    全名 客戶的全名，即根據最新客戶記錄的名字和姓氏
    電話號碼 可以是註冊期間提供的任何客戶的家庭或工作地點或手機號碼，也可以是客戶的最新記錄
    電子郵件 根據最新客戶記錄，客戶註冊的主要電子郵件地址
    聯絡人編號 客戶的聯繫ID。可以是主要或輔助聯繫人ID。
    CC4數字 客戶的信用卡末4位數字，以後註冊或添加到客戶帳戶中
    到期月份和年份 客戶的MM和YYYY格式的註冊信用卡的到期月份和年份
    禮品卡PIN碼 禮品卡的禮品卡密碼
  4. 如果搜索結果僅產生一個帳戶，則顯示客戶的“帳戶摘要"屏幕以查看360度客戶信息：

  5. 如果找到多個帳戶，則會在屏幕上顯示具有匹配信息的帳戶列表以及相應的客戶信息，例如渠道合作夥伴，帳戶名稱，電子郵件，CP客戶ID，SP帳戶ID，狀態，服務計劃和上次登錄日期帳戶的“搜索條件"屏幕下方。

    單擊任一客戶' 從搜索結果中的帳戶詳細信息導航到客戶帳戶摘要屏幕以查看 360 度客戶信息。
