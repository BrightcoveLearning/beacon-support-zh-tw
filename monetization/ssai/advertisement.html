概述

要創建廣告配置，您需要執行以下操作：

  • 從您的廣告服務提供商處獲取廣告標籤服務器 URL。這將在 Video Cloud Studio 中用於創建您的 SSAI 廣告配置。
  • 在 Video Cloud Studio 中，創建您的 SSAI 廣告配置。任務完成後，您將復制廣告服務器 ID 以在 Brightcove Beacon Classic 中創建廣告配置。
  • 在 Brightcove Beacon Classic 中創建您的廣告配置，然後您就可以將其應用於您的視頻內容。

創建 SSAI 廣告配置 ID

按照以下步驟創建 SSAI 廣告配置：

  1. 在 Video Cloud Studio 中，導航到管理 → 服務器端廣告設置 .
  2. 來自創建廣告配置下拉，選擇點播
  3. 為廣告配置添加名稱。
  4. 選擇您的廣告響應 - 建議使用 VMAP 或 VAST。如果您使用 VAST，則需要向視頻添加提示點以指定廣告插播位置應出現的位置。
  5. 粘貼您的廣告代碼。
  6. 點擊保存。
  7. 在表中找到您的新廣告配置並複制您的廣告配置 ID。您將需要此值用於 Beacon 廣告配置。

創建廣告配置

要創建廣告配置，請按照下列步驟操作：

  1. 在 Brightcove Beacon Classic 中，在頂部導航中選擇廣告 .
  2. 點擊添加新配置 .
  3. 命名廣告配置。
  4. 為了提供者選擇Brightcove SSAI
  5. 粘貼在SSAI 廣告配置 ID在上面的步驟中復製到VOD 廣告配置 ID .
  6. 點擊創建新廣告 .
