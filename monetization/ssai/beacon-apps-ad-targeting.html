介紹

客戶端信息從應用程序傳遞到SSAI，然後SSAI將此數據包含在客戶的VAST代碼中，然後VAST代碼可讓客戶根據此數據來定位廣告。這樣做的目的是增加客戶廣告庫存的價值，從而增加他們業務的廣告收入。

在 Studio 和 Beacon CMS 中，您可以設置如何將此數據傳遞到其廣告服務器或廣告提供商。

好處

改善廣告定位

  • 用戶會話 ID：當用戶打開應用程序時，Brightcove 在每台設備上生成一個會話 ID。這允許您為該會話設置頻次上限，排除競爭性廣告一起投放，並在該會話中按順序輪播廣告。這些定位功能對於直銷廣告和程序化廣告都是必要的。
  • 程序化廣告的更多數據：移動應用程序和CTV應用程序不支持Cookie，這是網絡上基於用戶/設備的廣告定位的主要方法。通過添加諸如設備ID之類的參數，您可以將這些有價值的信息傳遞給程序化廣告購買者，這將增強他們的廣告定位功能，從而帶來更高的每千次展示費用。

支持廣告庫存拆分

對於在Amazon，Roku和Samsung（在美國）上運行的應用，您需要使用該平台按70/30比例分配廣告資源/收入。這意味著 Brightcove Beacon 客戶通過 70% 的廣告庫存獲利，而平台則獲得剩餘 30% 的廣告庫存。通過這種增強的廣告定位，您可以在他們的所有應用中使用一個廣告代碼，並將設備參數傳遞給您的廣告服務器。在廣告服務器內部，您將實現業務邏輯，以按設備劃分廣告資源。平台還需要傳遞 LAT - 有限的廣告定位 - 用於廣告庫存拆分，這是我們現在發送到您的廣告服務器的參數。

用戶隱私改進

  • LAT“受限廣告定位"-允許Beacon應用程序的用戶根據用戶行為選擇退出廣告定位。
  • 用戶可以隨時重置我們分配給設備的唯一設備 ID。

實現客戶端參數

無論您是通過 VOD 還是 Live 使用廣告定位，您都需要構建將用於兩者的查詢字符串。構建查詢字符串時請遵循以下準則：

  • 請求廣告時，客戶端參數將傳遞給廣告服務器。
  • 客戶端參數可以在 Playback API 中傳遞（在 VMAP/Manifest 入口點中提供）。這些將以下列形式提供給 SSAI 基礎設施{{url.xxx}}宏在哪裡xxx對應參數鍵。
  • 粘貼您的廣告代碼，附加您的廣告定位參數查詢字符串： 
    variablename1={VALUEMACRO1}&variablename2={VALUEMACRO2}& ...
  • 例子： 
    plarform={PLATFORM}&os={OS}&appid={APP_ID}&did={RDID}&dt={IDTYPE}&lat={IS_LAT}&sid={SID}

請務必構建所需的廣告定位查詢字符串，因為您將在下面的實施步驟中用到它。

點播步驟

視頻雲步驟 - VOD

  1. 在 Video Cloud Studio 中，導航到管理 → 服務器端廣告設置 .（有關詳細信息，請參閱實施 VOD SSAI文檔。）
  2. 來自創建廣告配置下拉，選擇點播
  3. 為廣告配置添加名稱。
  4. 選擇您的廣告響應 - 建議使用 VMAP 或 VAST。如果您使用 VAST，則需要向視頻添加提示點以指定廣告插播位置應出現的位置。
  5. 粘貼到您的廣告代碼中，附加您的廣告定位參數查詢字符串。
  6. 點擊保存。
  7. 在表中找到您的新廣告配置並複制您的廣告配置 ID。您將需要此值用於 Beacon 廣告配置。

信標步驟 - VOD

  1. 打開 Beacon Classic UI 並導航到廣告標籤。
  2. 點擊添加新配置 .
  3. 命名廣告配置。
  4. 為了提供者選擇Brightcove SSAI .
  5. 粘貼根據上述部分構建的廣告定位 URL。
  6. 粘貼在SSAI 廣告配置 ID在上面的步驟中復製到VOD 廣告配置 ID .
  7. 點擊創建新廣告 .

現場步驟

視頻雲步驟 - 直播

  1. 在 Video Cloud Studio 中，導航到管理 → 服務器端廣告設置 .（有關詳細信息，請參閱實施 VOD SSAI文檔。）
  2. 來自創建廣告配置下拉，選擇居住
  3. 為廣告配置添加名稱
  4. 選擇廣闊的您的廣告響應
  5. 粘貼到您的廣告代碼中，附加您的廣告定位參數查詢字符串。
  6. 點擊保存。
  7. 在表中找到您的新廣告配置並複制您的廣告配置 ID。您將需要此值用於 Beacon 廣告配置。

信標步驟 - 現場直播

  1. 打開 Beacon Classic UI 並轉到廣告標籤。
  2. 點擊添加新配置 .
  3. 命名廣告配置。
  4. 為了提供者選擇Brightcove SSAI
  5. 粘貼根據上述部分構建的廣告定位 URL。
  6. 粘貼在SSAI 廣告配置 ID在上面的步驟中復製到VOD 廣告配置 ID .
  7. 點擊創建新廣告 .
