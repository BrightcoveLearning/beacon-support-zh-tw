概述

服務器端廣告嵌入在視頻流中，因此它們無法被瀏覽器中的廣告攔截器攔截。它適用於受DRM保護的內容和不受保護的內容。Brightcove Beacon支持SSAI的VOD。

以下是添加服務器端廣告的步驟：

  1. 在Video Cloud Studio中，創建廣告配置
  2. 設置自定義字段值為您的視頻
  3. 將您的視頻添加到播放列表
  4. 在Brightcove Beacon中，導入您的視頻和播放列表

最佳實務

在將服務器端廣告添加到VOD內容時，Brightcove建議遵循以下最佳做法：

  • 在Video Cloud中創建廣告配置
  • 使用視頻雲中的自定義字段將廣告配置分配給視頻
  • 即使您可以在Brightcove Beacon中為視頻分配廣告配置，還是建議在Video Cloud中進行配置
  • 僅使用廣告 Brightcove Beacon中的“現場"部分，用於直播活動和頻道

製作廣告配置

在這裡，您將創建SSAI廣告配置，以定義在Brightcove Beacon VOD資產中顯示的廣告。

  1. 登錄到Video Cloud Studio。
  2. 在導航標題中，單擊行政然後選擇服務器端廣告設置。將顯示所有廣告配置的列表。
  3. 要創建新的廣告配置，請點擊創建廣告配置按鈕。
  4. 從下拉菜單中選擇視頻點播作為類型。
  5. 檢查以確保您看到了服務器端廣告設置形式。
  6. 給您的廣告配置一個名稱。
  7. 選擇廣告服務器響應：
    廣告服務器響應 細節
    浩瀚 3.0 數字視頻廣告投放模板
    DFP廣告管理系統VMAP 視頻多廣告播放列表
    DFP（廣告規則） DFP / Google for Publishers的專有格式
    智能 XML FreeWheel的專有格式

    請與您的廣告運營經理核實以選擇正確的值。
  8. 輸入廣告代碼（服務器網址）。
  9. 點擊保存。
  10. 複製廣告配置的ID，以備日後配置VOD資產時使用。

設置自定義字段值

在本節中，您將學習如何使用必要的設置來配置VOD資產，以在Brightcove Beacon中顯示SSAI。

  1. 在Studio中，轉到媒體模塊。
  2. 單擊您要添加SSAI的視頻。
  3. 向下滾動到自定義字段部分。
  4. 點擊編輯按鈕。
  5. 向下滾動並找到信標.rights.0.adConfiguration字段，然後粘貼您之前創建的廣告配置的 ID。
  6. 繼續向下滾動並在beacon.rights.0.type現場寫影音將此視頻點播分類為“廣告視頻點播"。
  7. 轉到底部自定義字段部分，然後點擊保存按鈕。
    默認情況下，您最多可以添加 3 個 SSAI ID 和類型配置。自定義字段部分。
  8. 在頂部菜單中，單擊返回影片。

將視頻添加到播放列表

在本部分中，您將VOD資產添加到播放列表中。

  1. 在Studio中，返回到媒體模塊。
  2. 點擊您視頻的複選框。
  3. 點擊更多的從主菜單中選擇按鈕。
  4. 從下拉菜單中選擇添加到播放列表。
  5. 選擇要添加視頻的播放列表。
  6. 點擊添加按鈕。
  7. 將您的VOD資產添加到播放列表後，轉到Brightcove Beacon平台。

導入視頻和播放列表

在Brightcove Beacon中，您可以從Video Cloud提取/導入視頻和播放列表。

  1. 登錄到您的Brightcove Beacon帳戶。
  2. 在頂部菜單中，選擇工具，這是扳手圖標。
  3. 在左側導航中，選擇攝取。
  4. 點擊更新Brightcove視頻按鈕和/或更新Brightcove播放列表按鈕。
  5. 您來自 Video Cloud 的視頻和播放列表現在應該位於 Brightcove Beacon 中。在頂部菜單中，單擊播放清單標籤。
  6. 選擇您的播放列表以查看其中包含的視頻。
  7. 檢查以確保您可以在左目錄中看到與該播放列表關聯的所有VOD資產。
  8. 在Brightcove Beacon頂部菜單中，單擊電影標籤。
  9. 單擊您的VOD資產。
  10. 點擊權利與日程安排標籤。
  11. 在裡面權利類型菜單，您應該看到廣告-AVOD已選擇。
  12. 在裡面廣告配置菜單中，您應該會看到之前創建的廣告配置ID。
  13. 要立即在您的應用中查看您的視頻，您需要清除緩存。在頂部菜單中，單擊工具（扳手）按鈕，然後清除緩存。
  14. 在左側導航中，選擇快取。
  15. 選擇緩存清除按鈕。
  16. 打開您的Brightcove Beacon應用程序，然後單擊頁標籤上添加了您的播放列表。（圖像上的所有事件名稱和圖片僅供參考）。
    要在 Brightcove Beacon 應用程序中可視化您的播放列表，您必須先將播放列表添加到 Brightcove Beacon 頁面。要了解有關如何向主頁添加播放列表的更多信息，請訪問以下內容文件 .
  17. 單擊您配置了服務器端廣告的電影。
  18. 點擊玩按鈕，您應該會看到視頻播放廣告。
