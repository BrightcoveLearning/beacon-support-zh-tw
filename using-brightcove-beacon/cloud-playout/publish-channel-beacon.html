本主題中描述的某些操作在 Beacon 中執行，而其他操作在 Video Cloud 中執行。

介紹

創建 Cloud Playout 頻道後，您需要執行三個步驟才能使其成為 Beacon 應用程序的一部分。這些將在下面的部分中詳細說明。

獲取信標頻道的信息

在繼續之前，您需要獲取設置 Beacon 頻道和播放列表所需的信息。

  1. 通過單擊頻道列表中的頻道，轉到頻道詳細信息頁面。
  2. 如果您還沒有這樣做，請單擊 啟用激活通道。看使用頻道操作更多細節。
  3. 頻道激活後，使用主導航轉到視頻雲媒體模塊。
  4. 您應該會找到與您的 Cloud Playout 頻道同名的視頻。這將是一個沒有圖像的實時視頻：
  5. 單擊視頻名稱以打開詳細信息。
  6. 保存兩個視頻標識和頻道編號（後面的長字母數字字符串雲播放：在標籤中）。

創建信標通道

此步驟當前必須在信標 .

要在 Brightcove Beacon 中創建頻道，請按照下列步驟操作：

基本數據

  1. 登錄 Beacon Classic。
  2. 在頂部導航中，選擇頻道數。
  3. 點擊新增頻道按鈕。
  4. 在裡面基本數據標籤，定義一個名稱為您的頻道。
  5. 輸入 Cloud Playout 頻道 ID：

    要查找 Cloud Playout 頻道 ID，請參閱獲取 Beacon 頻道的信息 .

  6. 在裡面內容分級下拉菜單中，根據您的頻道流式播放國家/地區的內容分級系統選擇頻道的內容分級。在示例圖片中，它是針對美國的電視台。
  7. 在裡面類型詳情部分中，單擊體裁下拉菜單為您的頻道選擇類型。（例子：恐怖，戲劇等。）
  8. 如果找不到適合您頻道的流派，則可以通過點擊創建體裁按鈕。
    9. 這子流派下拉菜單此時不起作用。

文本數據

這文本數據部分允許您為頻道添加可顯示的文本。

  1. 選擇文本數據標籤。
  2. 在裡面標題字段，為您的頻道定義標題。
  3. 在裡面描述字段，為您的頻道定義說明。
  4. 在裡面概要字段，為您的頻道定義摘要。
    5. 您可以添加一個文本數據您將在頻道中使用的每種不同語言的部分。

流

在流部分，您將為您的頻道設置流信息

  1. 在流部分中，選擇亮灣作為提供者然後點擊添加新流：
  2. 添加 Steams 信息：
    • 流視頻 ID：輸入頻道直播的視頻 ID
    • 要發布的設備：單擊選擇/取消選擇所有設備，然後刪除您不想發佈到的任何設備
    • 流類型：選擇居住
    • 數字版權管理類型：選擇您正在使用的 DRM 類型或打開
    • 流格式：選擇 HLS

權限和調度

  1. 在裡面權限和調度部分，添加以下信息：
    • 地位：選擇已發布
    • 允許下載：選擇是否允許觀眾下載內容（大多數情況下，答案是否定的）
    • 權利類型：選擇免費或適合您內容的任何內容
    • 開始時間：使頻道可用的時間（選擇過去的日期以使其立即可用）
    • 時間結束：頻道不可用的時間（如果您希望它無限期地運行，您可以在將來設置它）
    • 最大流：您不需要輸入任何內容 - 將其保留為 0
    • 要發布的設備：單擊選擇/取消選擇所有設備，然後刪除您不想發佈到的任何設備
    • 允許的地點：進入世界或您允許查看的區域
    • 拒絕地點：留空或進入不允許查看的區域

創建頻道

  1. 輸入所有必要的設置後，單擊創建新頻道

將頻道添加到播放列表

因為 Beacon 頁面使用播放列表對於內容，您需要將 Cloud Playout 頻道添加到播放列表。播放列表可以包含其他視頻，但通常只包含頻道，以便包含頁面內容。看創建播放列表詳情。

將頻道添加到 Beacon 頁面

將 Cloud Playout 頻道放入 Beacon 應用程序的最後一步是最簡單的一步：您需要做的就是將包含頻道視頻的播放列表添加到頁面。看創建頁面詳情。
