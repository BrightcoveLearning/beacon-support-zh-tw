Cloud Playout 模塊使廣播公司和媒體發布商能夠使用 VOD 視頻和直播流的集合，然後將它們作為線性頻道進行流式傳輸。頻道可以流式傳輸到 Brightcove Live、Amazon S3 或指定的 RTMP URL。

要訪問 Cloud Playout 模塊，請單擊雲播出在導航菜單中。

將顯示已創建的所有Cloud Playout頻道的列表（如果有）。

該頁面將顯示以下信息：

  • 頻道名稱 -頻道名稱
  • 開始時間 - 頻道的開始時間
  • 時間結束 -頻道的結束時間
  • 循環播放 -指示程序是否將循環
  • 目的地 -頻道的目標位置
  • 創建日期 -頻道創建的日期
  • 最近更新時間 -頻道上次更新的日期
  • 地位 - 頻道狀態
    • 草案 -頻道已創建但尚未激活
    • 創作 -頻道正在激活中
    • 預定 - 頻道處於活動狀態並準備好流式傳輸
    • 開始 - 開始流媒體的頻道啟動過程
    • 跑步 - 頻道正在積極流式傳輸
    • 停止 - 頻道流正在停止
    • 空閒 -頻道成功運行並已停止
    • 刪除中 -頻道正在刪除
    • 錯誤 -創建或流式傳輸頻道時發生錯誤

建立頻道

要創建新的 Cloud Playout 頻道，請按照以下步驟操作。

  1. 點擊雲播出在導航標題中。
  2. 點擊 ：
  3. 進入頻道鑑別細節：
    • 名稱 -頻道名稱。名稱必須包含英文字符，並且允許使用以下特殊字符： _ 。 : / = + - 
    • 描述 -頻道說明
    • 圖片網址 -頻道圖片文件的網址；意味著可以作為通道元數據的一部分檢索的通道圖像
  4. 進入頻道日程細節。
    • 循環播放內容 - 如果上，程序將循環播放，並且結束日期/時間將被啟用。程序仍然可以隨時修改，更改將反映在當前程序和隨後的循環中。見創建循環程序以下部分。
    • 廣告插入 -如果啟用，則允許在頻道播放期間展示廣告。如果選擇此選項，則不允許實時輸入，並且您只能輸出到Brightcove現場直播或者亞馬遜S3桶。看貨幣化內容 .
    • 開始日期/時間 - 頻道開始播放的時間。所有資產的開始時間將通過將其持續時間添加到此開始時間來計算。該時間只有在通道不處於運行狀態時才能更新。
    • 結束日期/時間 -如果未輸入任何值，並且循環播放內容如果未選擇，則播放結束後，流將結束。如果循環播放內容被選中，沒有結束日期/時間，播放流將一直播放到該頻道停止播放為止。這個時間可以隨時更新。
  5. 如果要使用實時輸入，請設置現場輸入設為開啟，然後輸入Slate URL，除非您要為平板使用默認的Brightcove MP4。每當由於編碼器停止或某些技術問題阻止實時流而導致實時輸入停止時，slate MP4 都會播放。注意Slate URL必須是始終可用的靜態URL（不允許使用短暫的CDN URL）。
  6. 進入頻道目的地細節。Cloud Playout同時支持2個輸出。例如，您可以將HLS輸出到S3存儲桶中，並將RTMP提要輸出到Brightcove直播頻道或第3方攝入點。注意合併Brightcove現場直播和實時MP因為不支持同時目的地。
    • 優化顯示 -控制輸出的比特率和尺寸
      • 行動與網路 -如果選中，則為高清輸出（RTMP或HLS單比特率）
      • 智能電視 - 如果選中，全高清輸出（RTMP 或 HLS 單比特率）
    • 輸出區域 -將在其中創建通道的區域。Brightcove建議選擇地理上最接近大多數觀眾的地區。
    • 輸出到 -選擇通道的輸出方法
      • Brightcove現場直播 -創建或使用以前創建的Brightcove直播頻道。創建新頻道時，頻道的區域將根據輸出區域那是被選中的。
      • RTMP - 選擇將流輸出到 RTMP 或 RTP 攝取點。RTP 不適用於第三方攝取點；您只能在以下情況下使用它廣告插入和Brightcove 現場活動已啟用。選擇時，一個網址字段將出現 - 輸入您的實時流媒體服務的 RTMP 輸入 URL。
      • 亞馬遜S3桶 -選擇將HLS輸出推送到Amazon S3存儲桶。您將需要輸入存儲桶名稱、區域、訪問密鑰和秘密密鑰。存儲桶應該是公共的和可寫的。
      請注意，輸出到Brightcove 現場活動將自動為您的帳戶生成可計費的實時時間。使用 Cloud Playout 不會產生額外費用，但會收取直播時間的正常費用。輸出到RTMP或Amazon S3不會產生可計費的實時小時數：
  7. 點擊 .新頻道將被創建，狀態為草案 .

向頻道添加內容

看將內容添加到 Cloud Playout 頻道有關如何將 VOD 內容、播放列表和直播活動添加到您的頻道的信息。

另請參閱向內容添加保險槓用於在VOD或現場活動項目中添加保險槓。

將視頻和現場活動添加到頻道後，單擊啟用激活頻道。必須先激活頻道，然後才能開始流式傳輸。見使用頻道操作想要查詢更多的信息。

筆記：要更改程序中視頻的順序，請單擊視頻並將其拖動到新位置。

當視頻被添加到頻道時，系統生成的播放列表將被創建並在媒體模塊中可見。

我們強烈建議您對 Cloud Playout 模塊中的頻道進行任何更改，並且不要以任何方式編輯生成的播放列表 .

在離開Video Cloud並進入Beacon之前，您將需要捕獲在設置Beacon頻道時所需的信息。看創建信標通道查看如何輸入有關 Cloud Playout 頻道的信息。

  1. 通過單擊頻道列表中的頻道，轉到頻道詳細信息頁面。
  2. 如果您還沒有這樣做，請單擊 啟用激活通道。看使用頻道操作更多細節。
  3. 激活頻道後，使用主導航轉到媒體模塊。
  4. 在視頻列表的頂部，您應該看到為您的Cloud Playout頻道創建的新直播視頻：

編輯頻道

對於處於DRAFT或SCHEDULED狀態的通道

要編輯草稿或預定的 Cloud Playout 頻道：

  1. 單擊頻道名稱以打開頻道詳細信息。
  2. 點擊
  3. 在對話框中進行適當的編輯，然後單擊

對於運行中的頻道

要編輯正在運行的頻道：

  1. 單擊頻道名稱以打開頻道詳細信息。
  2. 點擊
  3. 點擊
  4. 頻道設置對話框將打開 - 一些無法更改的選項將不可用。
  5. 進行更改並單擊

刪除頻道

要刪除 Cloud Playout 頻道，請單擊頻道名稱旁邊的複選框，然後單擊刪除 .
