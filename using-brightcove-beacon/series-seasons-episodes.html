    創建系列/季節/劇集

    在本主題中，您將學習如何創建具有相關季節和情節的系列。

    概覽

    一種系列是按特定順序顯示的相關視頻的集合，通常具有相同的主題。建立序列時，您必須定義序列、其季節及相關單集之間的關係。

    一種季節是一組覆蓋一定時間段的視頻。每個系列可以有一個或多個季節。

    一個插曲是個人視頻。你可以把這個想像成一部特定的電影

    一旦你定義了你的系列和賽季，並攝入你的劇集，它們就可以在你的 Brightcove Beacon 應用程式中使用。為此，請按照下列步驟操作：

    1. 系列 (布萊特灣燈塔)

      在布萊特灣信標，創建該系列。在這裡，您將為系列命名，並提供包括可用性和映像等資訊。

    2. 季節 (布萊特灣燈塔)

      在布萊特灣燈塔，創造與系列相關的季節。除了選擇賽季所屬的系列之外，您還會提供類似系列的資訊。

    3. 集數 (雲端影片)

      布萊特灣信標集內容的真相存在於影片雲端。

      在 Video Cloud Studio 中，使用提供給自訂欄位的值，將影片集指派給對應的季節。這些自訂欄位包括：

      • beacon.episode.number
      • beacon.episode.seasonNumber
      • beacon.episode.serieName

    4. 返回布萊特灣信標並攝取您的視頻集。

    建立序列

    若要建立序列，請依照下列步驟執行：

    1. 登錄到Brightcove Beacon並選擇程式。
    2. 在左側導航中，選擇系列。
    3. 點擊添加新系列按鈕。

    4. 在基本數據標籤，填寫以下內容：

      表格欄位名稱 說明/要求
      名字 系列的標識符，面向客戶的名稱將在後面的文字數據部分
      製作年份 系列製作的年份；此信息將出現在更多信息屏幕
      製作國家 該系列的生產國；此信息將出現在更多信息屏幕
      類別 系列的類型；請注意，您可以視需要建立類型：
      子類型 請勿完成，不使用此資料
      類型語言預覽 預覽的語言
      插曲排序類型 如何在應用程式使用者介面中排序集數
      演員陣容 下的元數據投將出現在更多信息屏幕。
    5. 單擊文本數據選項卡。
    6. 填寫標題（這是該系列的觀看者姓名），描述和概要。此數據將顯示在Brightcove Beacon應用程序中。
    7. 單擊非文本數據選項卡。
    8. 提供下列項目的影像 (符合下列需求)：
      • 人像海報（需要）
      • 風景海報（需要）
      • 縮圖
      • 寬幅橫幅（需要）

      如果未提供，則會從「人像海報」產生縮圖影像。

      影像 需求
      人像海報

      JPEG圖像（2：3）

      最小值 寬度：1000 像素 高度：1500 像素
      最大值 寬度：2000 像素 高度：3000 像素
      風景海報

      JPEG圖像（16：9）

      最小值 寬度：1000 像素 高度：563 像素
      最大值 寬度：4000 像素 高度：2250 像素
      縮圖 JPEG圖像（4：3）
      寬橫幅 JPEG圖像（21：9）
      最小值 寬度：1000 像素 高度：428 像素
      最大值 寬度：4000 像素 高度：1714 像素
    9. 關於的信息標籤標籤目前未使用，因此請勿在此標籤上輸入信息。
    10. 關於的信息元數據標籤目前未使用，因此請勿在此標籤上輸入信息。
    11. 點擊流標籤，您將在其中選擇該系列的預告片。下面的截圖將幫助您完成選擇該預告片的過程。
    12. 選取您的提供者。您很可能只有一個提供者，這些供應商會參考您的布萊特灣視訊雲端帳戶。
    13. 為了流視頻ID，從Studio獲得預告片的視頻ID，並將其放置在文本框中。
    14. 請確定串流類型設定為 [預告片 ]。
    15. 請注意， DRM 類型和流格式值不需要更改。
    16. 點擊權利和計劃標籤，如下所示：

    17. 新增下列欄位的值：

      欄位 必填 注意事項
      狀態 是 選取下列其中一種類型：
      • 已發佈
      • 未發佈
      權限類型 是 從選項中為您的序列選擇適當的值：
      • 廣告-AVOD
      • 免費
      • 訂閱-SVOD
      開始時間 是 排定一個開始日期您的系列可供使用
      結束時間 是 排定結束日期，當您的系列將不再出現
      最大串流 是 設定可以從同一帳戶觀看系列的最大串流數
      要發佈的裝置 是 新增您要發佈的裝置。選擇選擇/取消選擇所有設備查看可用設備列表
      允許的位置 是

      如果沒有限制進入 World ，請選取您想要串流內容的位置。

      如果您沒有指定至少 1 個允許的位置，則您可能會從您的應用程序看到以下消息：
      拒絕的位置 否 選取您不想串流內容的位置
    18. 請務必點擊創建新系列按鈕保存系列信息。

    您現在已建立您的系列。下一步是建立屬於此系列的一個或多個賽季。下一節將引導您完成該過程。

    建立季節

    若要建立賽季，請按照下列步驟操作：

    1. 登錄到Brightcove Beacon並選擇程式。
    2. 在左側導航中，選擇季節。
    3. 點擊新增季節按鈕。
    4. 在基本數據標籤，填寫以下內容：
      表格欄位名稱 說明/要求
      選取序列 下拉式清單將會填入您所建立的序列。選擇本賽季所屬的系列。
      名字 系列的標識符，面向客戶的名稱將在後面的文字數據部分
      季度編號 這必須是代表該系列中季節序列的整數。從影片雲端擷取集數影片時，會使用此號碼。
      製作年份 季節的產生年份；此信息將出現在更多信息屏幕。
      製作國家 季節產生的國家；此信息將出現在更多信息屏幕。
    5. 單擊文本數據選項卡。
    6. 填寫標題，描述和概要這個季節。此數據將顯示在Brightcove Beacon應用程序中。
      對於本賽季標題，您可以輸入類似第1季，但是如果系列中有不同季節的主題，則可能是更好的選擇。例如，在每個大洲都集中於一個大洲的一系列大洲上，這些季節的好名字應該是歐洲，非洲，亞洲等
    7. 單擊非文本數據選項卡。
    8. 針對您的季節，請提供下列項目的影像 (符合下列需求)：

    9. 關於的信息標籤標籤目前未使用，因此請勿在此標籤上輸入信息。
    10. 關於的信息元數據標籤目前未使用，因此請勿在此標籤上輸入信息。
    11. 點擊流標籤，您將在其中選擇本季的預告片。
    12. 選取您的提供者。您很可能只有一個提供者，這些供應商會參考您的布萊特灣視訊雲端帳戶。
    13. 按一下 [ 新增資料流 ] 按鈕。
    14. 為了流視頻ID，從Studio獲得預告片的視頻ID，並將其放置在文本框中。
    15. 請確定串流類型設定為 [預告片 ]。
    16. 請注意， DRM 類型和流格式值不需要更改。

    18. 新增下列欄位的值：

    您現在已經建立了您的其中一個賽季。如果需要，為該系列創建更多的季節。下一個步驟是將影片從影片雲端擷取到布萊特灣信標。下一節將引導您完成該過程。

    擷取單集

    雖然創建系列和相關的季節是在布萊特灣信標完成, 該系列的創建情節是在視頻雲工作室完成.您將會在每個集合影片的自訂欄位中指定該系列和季節的價值，然後將這些影片擷取到 Brightcove Beacon。

    請記住，布萊特灣信標集內容的真相存在於視訊雲端。當您從影片雲端擷取內容時，您在布萊特灣信標中對單集所做的任何變更都會被覆寫。

    先決條件

    繼續執行下列步驟之前，請確定您已完成下列項目：

    1. 登入視訊雲端工作室。

    2. 從頁面頂部的下拉菜單中，選擇上載。

    3. 上傳您的集合影片。

    4. 從頁面頂部的下拉菜單中，選擇媒體。

    5. 按照中指定的步驟更新劇集視頻圖像圖像及其規格文件。

    6. 在導覽標題中，依序按一下 [ 管理員 ] 和 [ 視訊欄位 ]。

      您必須是帳戶管理員才能存取 ADMIN 功能表。
    7. 建立擷取影片至布萊特灣信標所需的自訂欄位。有關詳細信息，請參見創建視頻云自定義字段文件。

    步驟

    若要將集合影片攝入布萊特灣信標，請依照下列步驟執行：

    1. 返回視頻雲端工作室。
    2. 從頁面頂部的下拉菜單中，選擇媒體。

    3. 按一下其中一集影片的標題，進入影片的屬性頁面。

    4. 向下滾動，直到您在右側看到自定義字段部分。
    5. 點擊編輯按鈕。
    6. 找出與單集相關聯的三個自訂欄位。
    7. 對於集數影片，請在對應的自訂欄位中輸入集數、季節編號和系列名稱。
    8. 請務必點擊保存在為視頻輸入適當的值後單擊“確定”按鈕。
    9. 在所有集數影片的自訂欄位中重複輸入值。
    10. 登入布萊特灣燈塔
    11. 點擊工具類選項 （扳手圖標）。

    12. 確保攝取在左側菜單上選擇選項。

    13. 點擊更新Brightcove視頻按鈕以提取新配置的劇集視頻。

    14. 要使視頻立即在您的應用中可用，請通過單擊以下按鈕清除Brightcove Beacon的緩存快取左側導航欄中的選項，然後點擊緩存清除按鈕。

      有關使用Brightcove信標工具選項的更多詳細信息，請參閱使用工具文件。

    此結論是建立系列及其相關的季節和劇集的資訊。
