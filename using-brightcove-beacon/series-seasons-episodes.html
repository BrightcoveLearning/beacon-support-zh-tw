概述

一種系列是按特定順序顯示的相關視頻的集合，通常具有相同的主題。創建系列時，您必須定義系列、其季節和相關劇集之間的關係。

一種季節是一組涵蓋一定時間段的視頻。每個系列可以有一個或多個季節。

一個插曲是一個單獨的視頻。您可以將其視為特定的電影。

一旦定義了系列和季節，並攝取了劇集，就可以在Brightcove Beacon應用中使用它們。為此，請按照下列步驟操作：

  1. 系列（Brightcove Beacon）

    在Brightcove Beacon中，創建系列。在這裡，您將為系列命名，並提供包括可用性和圖像在內的信息。

  2. 四季（Brightcove Beacon）

    在Brightcove Beacon中，創建與該系列相關的季節。除了選擇季節所屬的系列之外，您還將提供類似於系列的信息。

  3. 劇集（視頻雲）

    Brightcove Beacon 劇集內容的真相存在於 Video Cloud 中。

    在Video Cloud Studio中，使用提供給自定義字段的值將視頻劇集分配給相應的季節。這些自定義字段包括：

    • beacon.episode.number
    • beacon.episode.seasonNumber
    • beacon.episode.serieName

  4. 返回Brightcove Beacon並攝取您的視頻片段。

創建系列

要創建系列，請按照下列步驟操作：

  1. 登錄到Brightcove Beacon，然後選擇程式。
  2. 在左側導航中，選擇系列。
  3. 點擊添加新系列按鈕。

  4. 在基本數據標籤，填寫以下內容：

    表格欄位名稱 說明/要求
    名稱 系列的標識符，面向客戶的名稱在後面提供文本數據部分
    生產年份 該系列的製作年份；此信息將出現在更多信息屏幕
    生產國 製作該系列的國家；此信息將出現在更多信息屏幕
    體裁 該系列的流派；請注意，您可以根據需要創建流派：
    子流派 不完整，不使用此數據
    體裁語言預覽 預覽語言
    情節排序類型 如何在應用程序用戶界面中對劇集進行排序
    投 下的元數據投擲將出現在更多信息屏幕。
  5. 點擊文字數據標籤。
  6. 填寫 標題 （這是該系列面向觀眾的名稱）， 描述 和 概要.此數據將顯示在 Brightcove Beacon 應用程序中。
  7. 點擊非文字數據標籤。
  8. 提供以下圖片（符合以下要求）：
    • 人像海報（必需的）
    • 風景海報（必需的）
    • 縮略圖
    • 寬幅橫幅（必需的）

    如果未提供，則將根據人像海報生成縮略圖圖像。

    圖像 要求
    人像海報

    JPEG圖像（2：3）

    最低限度 寬度：1000像素 高度：1500px
    最大 寬度：2000像素 高度：3000像素
    風景海報

    JPEG圖像（16：9）

    最低限度 寬度：1000像素 高度：563px
    最大 寬度：4000像素 高度：2250像素
    縮略圖 JPEG圖像（4：3）
    寬幅橫幅 JPEG圖像（21：9）
    最低限度 寬度：1000像素 高度：428像素
    最大 寬度：4000像素 高度：1714像素
  9. 有關的信息標籤此時不使用選項卡，因此不要在此選項卡上輸入信息。
  10. 有關的信息元數據此時不使用選項卡，因此不要在此選項卡上輸入信息。
  11. 點擊溪流標籤，您將在其中選擇該系列的預告片。以下屏幕截圖將幫助您完成選擇該預告片的過程。
  12. 選擇您的提供者 .您很可能只有一個提供商，該提供商引用您的 Brightcove Video Cloud 帳戶。
  13. 為了流視頻ID，從Studio獲得預告片的視頻ID，並將其放置在文本框中。
  14. 確保流類型被設定為 預告片.
  15. 點擊權利與日程安排標籤，如下所示：

  16. 為以下字段添加值：

    場地 必要 筆記
    地位 是的 選擇以下類型之一：
    • 已發表
    • 未發表
    權限類型 是的 從選項中選擇適合您的系列的值：
    • 廣告-AVOD
    • 自由
    • 訂閱 - 點播
    開始時間 是的 安排開始日期您的系列將可用
    時間結束 是的 安排一個結束日期，當您的系列不再可用時
    最大流 是的 設置同一個賬號可以觀看的最大流數
    要發布的設備 是的 添加您要發佈到的設備。選擇選擇/取消選擇所有設備查看可用設備列表
    允許的位置 是的

    如果沒有限制，請選擇要流式傳輸內容的位置 世界.

    如果您未指定至少一個允許的位置，那麼您可能會在您的應用中看到以下消息：

您現在已經創建了您的系列。下一步是創建屬於該系列的一個或多個季節。下一部分將引導您完成該過程。

創造季節

要創建一個季節，請按照下列步驟操作：

  1. 登錄到Brightcove Beacon，然後選擇程式。
  2. 在左側導航中，選擇季節。
  3. 點擊新增季節按鈕。
  4. 在基本數據標籤，填寫以下內容：
    表格欄位名稱 說明/要求
    選擇系列 下拉列表將填充您創建的系列。選擇本賽季所屬的系列。
    名稱 系列的標識符，面向客戶的名稱在後面提供文本數據部分
    季號 這必須是一個整數，表示系列中的季節序列。從 Video Cloud 中提取劇集視頻時將使用此數字。
    生產年份 生產季節的年份；此信息將出現在更多信息屏幕。
    生產國 生產季節的國家；此信息將出現在更多信息屏幕。
  5. 點擊文字數據標籤。
  6. 填寫 標題， 描述 和 概要 本賽季。此數據將顯示在 Brightcove Beacon 應用程序中。
    對於本賽季 標題，您可以輸入一個名稱，如 第1季，但如果一個系列有不同季節的主題，那可能是更好的選擇。例如，一個關於大陸的系列，每個季節都集中在一個大陸上，季節的好名字將是 歐洲， 非洲， 亞洲， 等等。
  7. 點擊非文字數據標籤。
  8. 根據您的季節，提供以下圖片（符合以下要求）：
    • 人像海報（必需的）
    • 風景海報（必需的）
    • 縮略圖
    • 寬幅橫幅（必需的）

    最低限度 寬度：1000像素 高度：563px
    最大 寬度：4000像素 高度：2250像素

    最大 寬度：4000像素 高度：1714像素

  10. 有關的信息元數據此時不使用選項卡，因此不要在此選項卡上輸入信息。
  11. 點擊溪流標籤，您將在其中選擇本季的預告片。
  12. 選擇您的提供者 .您很可能只有一個提供商，該提供商引用您的 Brightcove Video Cloud 帳戶。
  13. 點擊添加新的流按鈕。
  14. 為了流視頻ID，從Studio獲得預告片的視頻ID，並將其放置在文本框中。
  15. 確保流類型被設定為 預告片.
  16. 點擊權利與日程安排標籤，如下所示。

  17. 為以下字段添加值：

    場地 必要 筆記
    地位 是的 選擇以下類型之一：

您現在已經創建了一個季節。如果需要，可以為該系列創造更多的季節。下一步是將視頻雲中的劇集視頻攝取到 Brightcove Beacon。下一部分將引導您完成該過程。

攝取劇集

雖然在 Brightcove Beacon 中創建系列和相關季節，但在 Video Cloud Studio 中創建系列劇集。您將在每個劇集視頻的自定義字段中分配有關係列和季節的值，然後將這些視頻攝取到 Brightcove Beacon。

請記住，Brightcove Beacon 劇集內容的真相存在於 Video Cloud 中。當您從 Video Cloud 攝取內容時，您在 Brightcove Beacon 中對劇集所做的任何更改都將被覆蓋。

先決條件

在繼續執行以下步驟之前，請確保您已完成以下操作：

  1. 登錄到Video Cloud Studio。

  2. 從頁面頂部的下拉菜單中，選擇上載。

  3. 上傳您的劇集視頻。

  4. 從頁面頂部的下拉菜單中，選擇媒體。

  5. 更新您的劇集視頻圖像圖像及其規格文檔。

  6. 在導航標題中，單擊行政然後視頻場。

    您必須是帳戶管理員才能訪問行政菜單。
  7. 創建將視頻攝取到Brightcove Beacon所需的必需自定義字段。有關詳細信息，請參見創建視頻云自定義字段文檔。

腳步

要將劇集視頻攝取到 Brightcove Beacon，請按照以下步驟操作：

  1. 返回到Video Cloud Studio。
  2. 從頁面頂部的下拉菜單中，選擇媒體。

  3. 單擊其中一集視頻的標題，以進入視頻的屬性頁面。

  4. 向下滾動，直到您在右側看到自定義字段部分。
  5. 點擊編輯按鈕。
  6. 找到與情節關聯的三個自定義字段。
  7. 對於情節視頻，在相應的自定義字段中輸入情節編號，季號和系列名稱。
  8. 請務必點擊保存在為視頻輸入適當的值後單擊“確定"按鈕。
  9. 在所有插曲視頻的自定義字段中重複輸入值。
  10. 登錄到Brightcove Beacon。
  11. 點擊工具選項 （扳手圖標）。

  12. 確保攝取在左側菜單上選擇了選項。

  13. 點擊更新Brightcove視頻按鈕以提取新配置的劇集視頻。

  14. 要在您的應用程序中立即提供視頻，請單擊緩存左側導航欄中的選項，然後單擊緩存清除按鈕。

    有關使用 Brightcove Beacon Tools 選項的更多詳細信息，請參閱使用工具文檔。
