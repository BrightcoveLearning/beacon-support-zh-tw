使用工具

在本主題中，您將學習工具燈塔信標部分。

工具介紹

在Brightcove Beacon中扳手圖標（標記為1）代表一組工具。工具的選項是攝取和緩存 (2).本文檔探討了這兩個選項。

攝取

在“攝取"組中，您具有三個工具：

  • 更新 Brightcove 視頻：如果您已將視頻添加到“視頻雲"目錄中，或更改了現有視頻，則此按鈕會將新添加或更新的視頻同步到Brightcove Beacon。
  • 更新 Brightcove 播放列表：如果您已將播放列表添加到您的Video Cloud目錄中，或更改了現有的播放列表，則此按鈕會將新添加或更新的播放列表同步到Brightcove Beacon。
  • XML EPG - 從文件夾中獲取 XML：當您的 EPG（電子節目指南）需要更新時，此按鈕會檢索更新的 XML 文件。您將獲得一個用於放置文件的 S3 帳戶。
您還可以與 Brightcove 支持人員合作，創建一個自動執行攝取的計劃。

注意事項

請注意，並非在Studio中輸入的有關特定視頻的所有數據都會自動提取到Brightcove Beacon中。例如，以下內容未從Video Cloud提取到Brightcove Beacon中：

  • 可用時間：安排時間在 可用性 視頻屬性的一部分未攝取到 Brightcove Beacon。日期將是，但不是時間。
  • 地理過濾：地理過濾信息設置在 可用性 視頻屬性的一部分未攝取到 Brightcove Beacon。
上面列出的問題可以通過創建一個包裹在 Brightcove Beacon 中，通過自定義字段在攝取時將視頻分配給該包。見創建訂閱包和使用訂閱包文檔以獲取更多信息。

另請注意，並非Brightcove Beacon中提到的每個字段都具有關聯的自定義字段。例如 生產國 和 子流派 沒有與之關聯的相應自定義字段。

快取

在“緩存"組中，您有兩個工具：

  • 緩存清除：在 Brightcove Beacon 中進行更改時，單擊此按鈕可將這些更改滾動到當前配置中。此選項將除圖像之外的所有更改拉入當前配置。
  • 影像快取：如果您在Brightcove Beacon中更新任何圖像，請單擊此按鈕以將圖像更改滾動到當前配置。

不應隨意執行此過程，因為在生產中過於頻繁地執行此操作會對可伸縮性和性能產生不利影響。
