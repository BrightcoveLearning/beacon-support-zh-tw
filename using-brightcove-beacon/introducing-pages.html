頁面內容

    介紹頁面

    在本主題中，將向您概述使用頁數在Brightcove Beacon應用程序中。頁面包含使用者將看到的功能和視訊內容應用程式。

    頁面類型

    當您建立或編輯頁面時，「頁面類型」選項會提供下列項目：

    頁面類別

    頁面類型可分為三個類別。每個類別和詳細資料如下：

    1. 不可設定
      • 您選擇要包含在應用程式中的頁面，或不包含在應用程式中。
      • 文件包括不可配置的頁面提供了更多詳細信息，包括如何將這些類型的頁面之一添加到您的應用程序。
      • 不可配置頁面的示例是設定值和註冊，如下所示：
    2. 螢幕
      • 保存您的視頻內容，在旋轉木馬，列表等的頁面。
      • 高度可配置的使用塊，版面和播放清單。這裡顯示兩個例子：
      • 螢幕-英雄 _ 影像
        • 一種特殊類型的屏幕頁面，但允許在視頻內容後面的圖像。
        • 需要上傳不同平台的橫向和海報方向圖像，例如網絡（橫向）和移動（縱向）。
        • 的“英雄”屏幕頂部的區域是不可覆蓋的，並且不可配置：
    3. 重新導向
      • 重新導向頁面類型僅適用於 Web 應用程式。
      • 重定向空白在新標籤頁中打開網頁。
      • 重定向自我在同一標籤中打開網頁。
      • 在網址上指定網址外部重定向選項卡。

    頁面類型定義

    下表顯示頁面類型，包括說明及其用途。

    頁面類型 描述 在用戶界面中添加塊
    （是/否）    		 採用
    （必需/推薦/可選）
    自訂下載 不支援 否 N/A
    自訂退出 提供結束應用程式的功能。 否 需要在某些平台上，主要是智能電視。
    自訂登出 提供登出應用程式的功能。登出後，使用者會留在登陸頁面上，或讓使用者成為匿名使用者。 否 如果您已註冊用戶，則必須符合大多數移動平台的情況。
    自訂設定 提供對設定的存取。 否 強烈建議
    自訂登入 提供已註冊使用者登入 否 註冊使用者需要；如果您允許匿名瀏覽，則不需要
    自訂註冊 提供使用者註冊的方式 否 如果應用程式使用已註冊的使用者，則為必要；如果您允許匿名瀏覽
    EPG- 電子節目指南 (EPG) 會顯示直播電視節目指南 否 需要的是應用程序使用直播頻道
    直播播放器 播放即時視訊，並自動將方向變更為橫向。 否 如果你播放直播電視，建議使用 LivetV-Preview 屏幕，因為它向用戶呈現一個播放器，然後點擊/點擊將改變方向為橫向和播放視頻。
    即時預覽 使用播放器播放實時視頻作為較大頁面的一部分，然後點擊/點擊將以橫向播放視頻。 否 建議在直播播放器上使用
    我的所有項目 (_S) 提供一個包含兩個頁籤的頁面，其中一個顯示「最近關注」，另一個頁籤則顯示「我的最愛」。這將是空白頁，並顯示一條消息，提示您需要使用最近看過和收藏夾。 否 選用
    我的最愛 (_S) 包含使用者標示為我的最愛的視訊。 否 建議
    我的內容 (_S)-最近觀看 (_S) 包含最近 20 部觀看過的影片，並依時間順序列出。 否 建議
    重新導向-空白 僅限網頁；當功能表選項按一下 [重新導向] 時發生到新索引標籤。 否 選用
    重新導向-自我 僅適用於 Web; 當菜單選項點擊重定向發生到相同的選項卡。 否 選用
    螢幕- 在輪播或分頁中顯示群組的視訊。用於組織您的應用程序的內容。 是 必填
    螢幕-英雄 _ 影像 特殊類型的屏幕頁面，您可以在視頻內容後面添加圖像。 是 選用
    搜尋- 提供輸入畫面與輸入欄位，您可以在其中搜尋內容，然後顯示結果。 否 強烈建議

    設定螢幕頁面

    如本文檔前面所述，屏幕頁麵包含您的視頻，並提供用於顯示代表您的視頻的圖像的選項。在本節中，您將看到您的配置選項與這些選項如何在您的應用程序中的屏幕截圖。您將首先看到有關塊的信息，這是屏幕頁面上容器的最高級別。然後，您將看到佈局信息，它控制代表您的視頻的圖像的顯示方式。

    以黃色亮顯的是您可以使用的圖塊類型：

    單一區塊

    的單塊僅允許您向該塊添加一個佈局/播放列表組合。屏幕截圖顯示了單塊展示旋轉木馬。請注意，如果您嘗試將多個佈局/播放列表組合添加到單塊，或嘗試更改為單塊包含多個佈局/播放列表組合，您將收到錯誤消息：

    清單區塊

    的列表塊允許您向該塊添加多個佈局/播放列表組合。屏幕截圖顯示了列表塊其中包含三個旋轉木馬。

    表格 _ 圖塊

    的製表符允許您向該塊添加多個佈局/播放列表組合，但一次只能顯示一個，並且選項卡允許用戶選擇哪個可見。有兩個限制：

    • 表格塊內的佈局應該是相同的。
    • 一個標籤塊內，佈局應該是網格。
    屏幕截圖顯示了製表符可選擇播放列表。

    下拉式區塊

    的下拉塊允許您向該塊添加多個佈局/播放列表組合，但一次只能顯示一個，並且下拉菜單允許用戶選擇哪個可見。有兩個限制：

    • 下拉塊內的佈局應該是相同的。
    • 內部下拉塊，佈局應該是網格。
    屏幕截圖顯示了下拉塊（在黃色矩形內）和三個播放列表可供選擇。

    配置

    在所選佈局上方塊的所有例子是一個旋轉木馬。旋轉木馬在水平列表中顯示每個視頻的圖像。這不是您可以選擇的唯一佈局。可用的配置會以黃色反白顯示：

    將顯示每個佈局特性的示例，而不是每種佈局類型的屏幕截圖。上面以黃色突出顯示的類型是不同特徵的組合。

    旋轉木馬和標籤

    旋轉木馬以水平列顯示圖像，標籤覆蓋圖像的小底部。

    標示輸出

    標籤位於影像下方。

    特色內容

    圖像更大（以紅色為盒裝，正常旋轉木馬下面）。通常，但並非總是會在頁面頂端使用精選版面。

    橫式

    影像是橫向比例。

    海報

    圖像是海報（縱向）比例。

    網格 3 和網格 4

    這兩種類型的網格會以二維列/欄樣式顯示影像。格點後面的數字 (3 或 4) 表示影像的相對大小。它不會指示永遠顯示的列數。這個屏幕截圖顯示了一個 800 像素寬的瀏覽器，網格 3 有兩行三個圖像。由於影像較小，網格 4 正在顯示四個影像。

    下一個截圖仍然使用網格 3 和網格 4，但瀏覽器寬度為 1600px。網格 3 的影像現在是四個橫向，網格 4 版面中的五個影像可以放在螢幕上。再次，你可以看到網格 3 圖像更大。

    這就結束了圖塊和配置的選項。
