頁麵類型

當您創建或編輯頁面時，頁麵類型選擇提供以下內容：

頁麵類別

頁麵類型可以分為三類。每個類別和詳細信息如下：

  1. 不可配置
    • 您選擇是否包含在應用程序中的頁面。
    • 文件包括不可配置的頁面提供了更多詳細信息，包括如何將這些類型的頁面之一添加到您的應用程序。
    • 不可配置頁面的示例是設定值和報名，顯示在這裡：
  2. 屏幕
    • 輪播，列表等中包含您的視頻內容的頁面。
    • 高度可配置的使用塊，版面和播放清單。顯示了移動平台上的兩個示例，然後是 Web 應用程序示例：
    • 屏幕-hero_image
      • 一種特殊類型的屏幕頁面，但允許在視頻內容後面顯示圖像。
      • 需要同時上傳不同平台的風景和海報方向圖像，例如，網絡（風景）和移動設備（人像）。
      • 這“英雄”屏幕頂部的區域是非覆蓋的，不可配置：
  3. 重新導向
    • 重定向頁麵類型僅適用於Web應用程序。
    • 重定向空白在新選項卡中打開網頁。
    • 重定向自我在同一選項卡中打開網頁。
    • 指定網址外部重定向選項卡。

配置屏幕頁面

正如本文檔前面提到的， 屏幕 頁麵包含您的視頻並提供用於顯示代表您的視頻的圖像的選項。在本節中，您將看到配置選項以及這些選項在應用程序中的外觀的屏幕截圖。您將首先看到有關塊的信息，這是屏幕頁面上容器的最高級別。然後您將看到佈局信息，它控制如何顯示代表您的視頻的圖像。

積木

以黃色突出顯示的是您可以使用的塊類型：

這是每種類型的圖像：

單塊

這 單塊 只允許您向塊添加一個佈局/播放列表組合。屏幕截圖顯示了一個 單塊 顯示輪播。請注意，如果您嘗試將多個佈局/播放列表組合添加到一個 單塊，或嘗試更改為 單塊 包含多個佈局/播放列表組合，您將收到錯誤消息：

列表塊

這 列表塊 允許您向塊添加多個佈局/播放列表組合。屏幕截圖顯示了一個 列表塊 其中包含三個旋轉木馬。

tabed_block

這 標籤塊 允許您向塊添加多個佈局/播放列表組合，但一次只會顯示一個，選項卡允許用戶選擇哪個可見。有兩個限制適用：

  • 標籤塊內的佈局應該是相同的。
  • 在製表符塊內，佈局應為網格。
屏幕截圖顯示了一個 標籤塊 可選擇播放列表。

下拉塊

這 下拉塊 允許您向塊添加多個佈局/播放列表組合，但一次只會顯示一個，下拉菜單允許用戶選擇哪個可見。有兩個限制適用：

  • 下拉塊內的佈局應該相同。
  • 在下拉框內，佈局應為網格。
屏幕截圖顯示了一個 下拉塊 （在黃色矩形內），可選擇三個播放列表。

版面

在所選佈局上方的所有塊示例中，都是一個輪播。輪播在水平列表中顯示每個視頻的圖像。這不是您可以選擇的唯一佈局。可用的佈局以黃色突出顯示：

海報

網格3和網格4

這兩種類型的網格以二維行/列模式顯示圖像。網格後面的數字 3 或 4 表示圖像的相對大小。它沒有指示將始終顯示的列數。此屏幕截圖顯示了一個 800px 寬的瀏覽器，Grid 3 有兩行三個圖像。由於圖像較小，網格 4 顯示四張圖像。

下一個屏幕截圖仍然使用 Grid 3 和 Grid 4，但瀏覽器寬度為 1600px。Grid 3 圖像現在是四張，Grid 4 佈局中的五張圖像可以適應整個屏幕。再次您可以看到Grid 3圖像更大。
