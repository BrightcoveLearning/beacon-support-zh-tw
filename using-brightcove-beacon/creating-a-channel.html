    建立通道

    在本主題中，您將學習如何使用布萊特灣信標中的頻道索引標籤建立頻道。

    描述

    頻道是為了串流 24 小時直播活動，就像一般的電視頻道一樣。頻道的內容可以透過一組 VOD 產生，只要頻道有可用，或直接傳送即時事件，就能直播。有關如何創建直播活動的更多信息，請訪問Brightcove信標實時文檔。

    「基本資料」段落

    若要在布萊特灣信標建立頻道，請依照下列步驟執行：

    1. 登入布萊特灣燈塔
    2. 在頂部導航中，選擇頻道。
    3. 點擊新增頻道按鈕。
    4. 在裡面基本數據標籤，定義一個名稱為您的頻道。
    5. 在裡面內容分級下拉菜單中，根據您的頻道流式播放國家/地區的內容分級系統選擇頻道的內容分級。在示例圖片中，它是針對美國的電視台。
    6. 在裡面類型詳情部分，點擊體裁下拉菜單為您的頻道選擇類型。(範例：恐怖, 戲劇等..)
    7. 如果找不到適合您頻道的流派，則可以通過點擊創建體裁按鈕。
      8. 的子流派下拉菜單，目前不起作用。

    文字資料

    的文字數據部分可讓您為頻道添加可顯示的文本。

    1. 選取「文字資料」頁籤。
    2. 在裡面標題字段，為您的頻道定義標題。
    3. 在裡面描述字段，為您的頻道定義說明。
    4. 在裡面概要字段，為您的頻道定義摘要。
      5. 您可以添加一個文字數據部分，用於您將在頻道中使用的每種不同的語言。

    非文字資料

    的非文字數據部分中，您可以為頻道添加縱向和橫向圖像。您將提供縱向和橫向圖像。請確定影像符合下列需求：

    影像 需求
    人像海報 海報影像 (2:3)
    最小值 寬度：1000 像素 高度：563 像素
    最大值 寬度：4000 像素 高度：2250 像素
    風景海報 橫向影像 (16:9)：
    最小值 寬度：1000 像素 高度：563 像素
    最大值 寬度：4000 像素 高度：2250 像素

    1. 選擇非文字數據標籤。
    2. 添加一個人像海報通過在框部分中拖放圖片來獲取您頻道的圖片。
    3. 添加一個風景海報通過在框部分中拖放圖片來獲取您頻道的圖片。
    只要選擇想要的語言分頁，您就可以為頻道中的每種語言新增不同的「直向」和「橫向」影像組。

    標籤

    「標籤」標籤目前無法運作。

    中繼資料

    的元數據標籤目前不起作用。

    串流

    的流部分可讓您設置播放URL的選項和權限。

    1. 選擇流標籤。
    2. 選取您的提供者。您很可能只有一個提供者，這些供應商會參考您的布萊特灣視訊雲端帳戶。
    3. 按一下 [ 新增資料流 ] 按鈕。
    4. 在裡面流視頻ID字段中，粘貼“視頻雲直播”活動中的視頻ID。
    5. 在裡面要發布的設備部分中，添加您要發佈到的設備。選取 [ 選取/取消選取所有裝置 ] 以查看可用裝置的清單。
    6. 在裡面流類型選擇生活。
    7. 請注意， DRM 類型和流格式值不需要更改。

    權利與排程

    [ 權限與排程 ] 區段是您設定發佈狀態和可用性的地方。

    1. 選取 [ 權限與排程 ] 索引標籤。

    2. 確保狀態被設置為已發表為您的頻道。
    3. 如需「可用性詳細資訊」，請使用下列指南來完成資訊：
      欄位 必填 注意事項
      權限類型 是 根據您的營利模式選擇：
      • 自由
      • 廣告-AVOD：適用於透過廣告營利的頻道。
      • 訂閱-SVOD：透過訂閱賺取營利的頻道。
      開始時間 是 為您的頻道安排開始時間。
      結束時間 是 為您的頻道排定結束時間。
      最大串流 N/A 目前無法正常運作。
      要發佈的裝置 是 新增您想要發佈的裝置。選取「選取/取消選取所有裝置」以查看可用裝置的清單。
      允許的位置 是 選取您要串流內容的位置。
      拒絕的位置 否 選取您不想串流內容的位置。

      若要新增或編輯地理位置，請導覽至「設定」，然後瀏覽「位置」。
    4. 請務必點擊建立新頻道按鈕保存您的工作。

    將您的頻道新增至電子節目指南 (EPG)

    一旦您的頻道建立完成後，最後一步將是將其添加到布萊特灣信標 EPG 時間表。有關如何執行此操作的信息，請訪問創建電子節目指南（EPG）文檔。
