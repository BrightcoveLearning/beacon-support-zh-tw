    實施 VOD SSAI

    在本主題中，您將學習如何使用 Brightcove 信標的隨選視訊 (VOD) 來建立和管理伺服器端廣告插入 (SSAI)。

    概覽

    伺服器端廣告內嵌於影片串流中，因此瀏覽器中的廣告封鎖程式無法封鎖這些廣告。它同時適用於受 DRM 保護和不受保護的內容。布萊特灣信標支援 SSAI 的 VOD。

    以下是新增伺服器端廣告的步驟：

    1. 在Video Cloud Studio中，創建廣告配置
    2. 設置自定義字段值為您的視頻
    3. 將您的視頻添加到播放列表
    4. 在Brightcove Beacon中導入您的視頻和播放列表

    最佳實務

    在 VOD 內容中新增伺服器端廣告時，Brightcove 建議遵循以下最佳做法：

    • 在視訊雲端中建立您的廣告設定
    • 使用 Video Cloud 中的自訂欄位將廣告設定指派給影片
    • 即使您可以將廣告設定指派給布萊特灣信標的影片，建議您在 Video Cloud 中執行此操作
    • 僅使用廣告 Brightcove Beacon中的“直播”和頻道部分

    建立廣告組態

    您將在這裡建立一個 SSAI 廣告配置，以定義您在布萊特灣信標 VOD 資產中顯示的廣告。

    1. 登入視訊雲端工作室。
    2. 在導覽標題中，按一下 [ 管理 ]，然後選取 [ 伺服器端廣告設定 ]。將會顯示所有廣告設定的清單。
    3. 若要建立新的廣告組態，請按一下 [ 建立廣告組態 ] 按鈕。
    4. 從下拉菜單中選擇視頻點播作為類型。
    5. 請確認您看到的是「伺服器端廣告設定」表單。
    6. 為您的廣告設定命名。
    7. 選取廣告伺服器回應：
      廣告伺服器回應 詳細資料
      VAST 3.0 數字視頻廣告投放模板
      DFP廣告管理系統VMAP 視頻多個廣告播放列表
      DFP (廣告規則) DFP / Google for Publishers的專有格式
      智慧型 XML FreeWheel的專有格式

      請洽詢您的廣告營運管理員，以取得正確的值以選取。
    8. 輸入廣告代碼 (伺服器網址) 。
    9. 按一下「儲存」。
    10. 複製廣告設定的 ID，因為您稍後需要它來設定 VOD 資產。

    設定自訂欄位值

    在本節中，您將學習如何使用必要的設置來配置您的 VOD 資產，以在布萊特灣信標顯示 SSAI。

    1. 在Studio中，轉到媒體模塊。
    2. 按一下您要新增 SSAI 的視訊。
    3. 向下滾動到自定義字段部分。
    4. 點擊編輯按鈕。
    5. 向下滾動並找到beacon.rights.0.adConfiguration字段，然後粘貼您先前創建的廣告配置的ID。
    6. 繼續向下滾動並在beacon.rights.0.type現場寫影音將此視頻點播分類為廣告視頻點播。
    7. 轉到底部自定義字段部分，然後單擊保存按鈕。
      默認情況下，您最多可以添加3個SSAI ID，並在自定義字段部分。
    8. 在頂部菜單中，單擊返回影片。

    新增影片至播放清單

    在本節中，您將將您的 VOD 資產添加到播放列表。

    1. 在Studio中，返回到媒體模塊。
    2. 按一下影片的核取方塊。
    3. 點擊更多從主菜單
    4. 從下拉菜單中選擇添加到播放列表。
    5. 選擇您要新增影片的播放清單。
    6. 按一下「新增」按鈕。
    7. 將您的 VOD 資產新增至播放清單後，請前往布萊特灣信標平台。

    匯入視訊和播放清單

    在布萊特灣信標，您可以從視訊雲端擷取/匯入影片和播放清單。

    1. 登入您的布萊特灣信標帳號。
    2. 在頂部菜單中，選擇工具類，這是扳手圖標。
    3. 在左側導航中，選擇攝取。
    4. 點擊更新Brightcove視頻按鈕和/或更新Brightcove播放列表按鈕。
    5. 您從視訊雲端中的影片和播放清單現在應該位於布萊特灣信標。在頂部菜單中，單擊播放清單標籤。
    6. 選取您的播放清單，即可查看其中包含的影片。
    7. 請確認您可以在左側目錄中看到與該播放清單相關聯的所有 VOD 資產。
    8. 在Brightcove Beacon頂部菜單中，單擊電影標籤。
    9. 點擊您的 VOD 資產。
    10. 點擊權利和計劃標籤。
    11. 在裡面權利類型菜單，您應該看到廣告-AVOD已選擇。
    12. 在裡面廣告配置菜單中，您應該會看到之前創建的廣告配置ID。
    13. 若要立即在應用程式中查看您的影片，您必須清除快取。在頂部菜單中，單擊工具類（扳手）按鈕，然後清除緩存。
    14. 在左側導航中，選擇快取。
    15. 選擇緩存清除按鈕。
    16. 打開您的Brightcove Beacon應用程序，然後單擊頁標籤上添加了播放列表。（圖片上的所有活動名稱和圖片僅供例如用途）。
      若要在布萊特灣信標應用程式中視覺化您的播放清單，您必須先將播放清單新增至布萊特灣信標頁面。要了解有關如何向頁面添加播放列表的更多信息，請訪問以下內容文件資料。
    17. 按一下您已設定伺服器端廣告的影片。
    18. 點擊玩按鈕，您應該會看到視頻播放廣告。
