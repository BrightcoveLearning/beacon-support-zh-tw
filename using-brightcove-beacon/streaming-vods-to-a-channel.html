    將 VOD 串流至頻道

    在本主題中，您將學習如何將一組隨選視訊資產 (VOD) 做為即時串流串流至布萊特灣信標的頻道。

    概覽

    要將一組VOD作為實時流流式傳輸到Brightcove Beacon中的頻道，您必須遵循一系列步驟來創建視頻供稿並將其發送給第三方Cloud流媒體合作夥伴。

    目前，Brightcove不支持此功能，因此有必要使用外部第三方公司（如Frequency，Veset或Amagi）的服務來完成從一組VOD生成實時流的過程。

    該過程的高級視圖是：

    1. 標記您要在流中使用的視頻。
    2. 使用Social Syndication API創建所需視頻的XML視頻供稿。
    3. 將您的XML視頻供稿URL發送給您的第三方Cloud Streaming合作夥伴。
    4. 使用您的Cloud Streaming合作夥伴提供的URL在“媒體”模塊中創建遠程視頻。
    5. 通過視頻ID將遠程視頻與Brightcove Beacon連接。
    6. 將您的Cloud Streaming提供商提供的XML文件上傳到Amazon S3存儲桶以生成電子編程指南（EPG）

    本文件的其餘部分將提供上述每個步驟的詳細資料。

    標記將在流中的視頻

    您將使用標籤（元數據）標記要在視頻流中播放的視頻。

    1. 在Video Cloud Studio中，打開媒體模塊。（屏幕截圖中的數字指的是後面的步驟。）
    2. 選中您要在視頻流中使用的視頻前面的所有復選框。
    3. 點擊快速編輯位於視頻列表上方的按鈕。
    4. 在裡面快速編輯影片標題右側的面板中，要添加的標籤輸入框，輸入您選擇用來對流中想要的所有視頻進行分組的標籤。在此示例中，標籤Aug_Stream用來。
      將需要標記來填充XML模板信息。

    使用社交聯合API創建XML視頻供稿

    要將標記的VOD集作為實時流進行流處理，您需要創建一個Video Feed XML文件。該文件將包含您希望流式傳輸的視頻的所有信息。任何第三方Cloud Streaming合作夥伴（例如Frequency，Veset或Amagi）都需要此文件。以下提供了此過程的詳細信息：

    • 您將需要以下API開發工具：郵差要么失眠。在這個例子中郵差將會被使用。
    • 要創建視頻Feed XML文件，您將需要使用Brightcove社交聯合API，有關如何使用該API獲取XML URL的信息，請訪問社交聯合API文檔。
    • 大多數平台 API 的身份驗證是通過臨時的 OAuth2 訪問令牌。一些API改用永久API密鑰或策略密鑰。見文件資料有關詳細信息，請參見各個API。
    1. 在此示例中，打開您的API客戶端工具郵差用來。
    2. 選擇開機自檢來自API調用類型dropmenu。
    3. 複製社交聯合API調用： 
      https://social.api.brightcove.com/v1/accounts/your-account-id/mrss/syndications
    4. 將Social Syndication API調用粘貼到URL字段中，然後使用您自己的帳戶ID信息更改URL數據。
    5. 選擇身體標籤在您的API客戶端中。
    6. 複製以下社會聯合資源模板：

    8. 使用您自己的信息編輯社交聯合資源模板數據。
    9. 請記住，將您之前在Video Cloud中創建的標籤添加到“ include_filter”：“標籤：YourTag”領域。
    10. 點擊發送 API客戶端中的按鈕開機自檢呼叫。

    13. 複製API響應ID。
    14. 下一步是為數據添加模板。從下拉菜單中選擇放。
    15. 添加ID URL請求中最後一個API調用的響應。
    16. 點擊身體標籤。
    17. 選擇生的選項。
    18. 複製並粘貼以下內容之一為通用聯合提供的XML模板在文字部分。
    19. 請點擊發送使放 API調用。
      請記住，您將需要在您的有效令牌中授權>管理訪問令牌 API調用部分可以正常工作。有關如何創建OAuth 2.0令牌的更多信息，請閱讀OAuth API文檔。
    20. 製成後放調用，API響應將包含帶有您的視頻信息的模板以及syndication_url將進行更新，並準備交付給您的Cloud Streaming合作夥伴。

    將XML提要交付給您的Cloud Streaming合作夥伴

    一旦有了syndication_url如果視頻Feed已更新，則需要將其交付給Cloud Streaming合作夥伴（頻率，Veset，Amagi），因為URL包含XML文件。

    收到它之後，您的Cloud Streaming合作夥伴應為您提供一個流URL和一個新的XML文件，其中包含Brightcove Beacon電子編程指南。

    在Studio中添加遠程視頻

    收到頻道的流URL後，您需要將實時流作為視頻雲中的遠程視頻發送到Brightcove Beacon。

    1. 打開媒體視頻雲帳戶中的模塊。
    2. 在左側面板中，點擊+添加遠程視頻按鈕。
    3. 在裡面影片名稱字段中，為您的流視頻命名。
    4. 在裡面影片類型菜單中，選擇生活選項。
    5. 在裡面格式字段中，選擇HLS選項。
    6. 在裡面網址字段中，粘貼您的Cloud Streaming合作夥伴提供的流URL。
    7. 按一下「儲存」按鈕。
    8. 創建遠程視頻後，請在媒體模塊。
    9. 打開視頻信息部分，然後復制ID。

    將遠程視頻連接到Brightcove Beacon

    最後一步是將您剛剛在Video Cloud中創建的Remote Video流連接到Brightcove Beacon Channel。

    1. 打開Brightcove信標。
    2. 在上方菜單中，點擊頻道標籤。
    3. 點擊您想要的頻道。有關如何創建頻道的更多信息，請訪問渠道文件。
    4. 在頻道設置中，點擊流標籤。
    5. 在裡面流視頻ID字段，粘貼視頻ID從您的視頻雲遠程視頻。
    6. 請點擊更新頻道。
    7. 打開Brightcove Beacon Web應用程序進行測試，您應該在所需的頻道上看到VOD流。
    8. 最後一步是將第三方Cloud Streaming合作夥伴提供的XML文件上傳到Amazon S3存儲桶中。這將為頻道創建EPG。有關如何上傳和獲取XML文件的更多信息，請訪問EPG文檔。
