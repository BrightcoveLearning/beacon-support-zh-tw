為您的應用選擇語言

在預售過程中，您聲明了將應用程序翻譯成多種語言的意圖。翻譯的可能性包括應用程序字段和內容元數據。

應用翻譯是通過請求翻譯您的應用體驗中的應用字段來完成的。內容翻譯是您輸入到 CMS 中的元數據。它包括文本數據，如標題和描述以及包含文本的圖像。

定義您希望在 Brightcove Beacon 應用程序中使用的語言是入職流程的一部分。在入職流程之後，如果您選擇開始使用多種語言，或者希望添加已有翻譯的語言，請聯繫您的 Brightcove 支持人員。如需新語言翻譯，請聯繫 Brightcove 支持。

什麼是翻譯？

應用程序有兩部分可以翻譯/可以翻譯，它們是：

  • 應用程序中的字段名稱、按鈕和描述的翻譯可由您提供，也可由 Brightcove 為您翻譯。
  • 您定義的項目的翻譯，例如頁面名稱或視頻元數據，以及圖像由您輸入 Brightcove Beacon。

您的觀看者如何為應用選擇語言

您的查看者可以通過三種方式選擇要在應用程序中使用的語言。他們是：

  • 選擇語言中的設置頁
  • 設置瀏覽器的語言（如果適用）
  • 為您的設備設置語言（如果適用）

在瀏覽器中設置首選語言

如果查看者在瀏覽器中設置了首選語言，並且 Brightcove Beacon 應用程序支持該語言，則該應用程序將自動使用該語言。觀眾不必在設置標準頁面。

筆記：如果在實際的 Brightcove Beacon 應用程序中設置了語言，則會覆蓋瀏覽器設置。

定義視頻元數據語言

本地化視頻元數據的基本過程是：

  • 配置所需的語言Video Cloud Studio 的管理員模塊。
  • 在視頻屬性頁面中，選擇要為其輸入本地化元數據的語言。
  • 輸入本地化的元數據。

有關使用多語言元數據的更多信息，請參閱使用多語言元數據文檔。

使用多語言元數據的第一步是通過導航到 Video Cloud Studio 中的元數據選擇語言管理 → 語言設置 .您首先添加語言然後保存它，如下所示：
