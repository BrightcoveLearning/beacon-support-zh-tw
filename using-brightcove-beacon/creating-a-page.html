創建新頁面

在本主題中，您將學習如何創建新的Brightcove信標頁面。

概述

Brightcove Beacon 頁面是用戶選擇和觀看您的視頻內容的地方。輪播顯示您在Video Cloud Studio中定義的一組視頻或播放列表。使用 Brightcove Beacon 管理控制台，您可以創建一個帶有內容輪播的新頁面。

創建一個頁面

要創建新頁面，請按照下列步驟操作：

  1. 登錄到Brightcove信標。

  2. 在頂部導航中，選擇頁數。

  3. 選擇添加新頁面按鈕創建新頁面。

  4. 為了基本數據標籤中，輸入以下字段值：

    • 系統名稱 - 內部系統頁面名稱。這不會向用戶顯示，但用於生成友好的 URL。您將在後面的步驟中輸入一個可見的標題。
    • 命令 -主頁上的頁面順序
    • 體裁類型 -值：
      • 掌握 - 主頁
      • 孩子 -下屬頁面
    • 導航類型 -從頁面模板列表中選擇
    • 演講類型 -值：
      • 圖標文字 -在其中顯示帶有文字的圖標
      • 圖標 -顯示圖標
      • 文本 - 顯示文字
    • 屏幕佈局 - 值：（並非每個導航類型都有屏幕佈局）
      • 屏幕佈局標題 - 帶有標題的佈局
      • 沒有標題的屏幕佈局 - 沒有標題的佈局
    • 到家了嗎 -選擇這是否是主頁
  5. 接下來，我們將在頁面上添加一個塊。塊是您的播放列表的容器。某些導航類型沒有塊，例如登錄和註銷類型。我們選擇了一個導航類型價值為屏幕 .

  6. 選擇添加塊按鈕。當您選擇按鈕時，塊類型將出現在右側。

  7. 選擇所需的塊類型。在這種情況下，我們選擇列表塊。選擇添加塊按鈕與您想要的塊相關聯。

  8. 選擇添加佈局按鈕。當您選擇按鈕時，佈局類型將出現在右側。

  9. 選擇所需的佈局類型，無論是“特色"，“輪播"還是“網格"。在此示例中，我們選擇輪播景觀標籤佈局。

  10. 選擇添加播放列表按鈕。當您選擇該按鈕時，播放列表類型將出現在右側。

  11. 選擇一個播放列表。在此示例中，我們選擇添加播放列表與關聯的按鈕幽默播放列表。

    您使用 Video Cloud Studio 創建播放列表，一旦您同步到您的 Video Cloud 帳戶，它們就會顯示在此處。

  12. 我們可以看到幽默播放列表已添加到我們的輪播中。您可以拖放以重新排序項目。

  13. 為了文字數據，非文字數據，標籤和權利與日程安排標籤，請參閱通用配置設置文檔。

  14. 選擇創建新頁面按鈕來創建您的新頁面。你都準備好了。您應該會在 Brightcove Beacon 頁面上看到新的輪播。
