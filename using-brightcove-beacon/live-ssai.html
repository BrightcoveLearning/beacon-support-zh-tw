    實施即時 SSAI

    在本主題中，您將學習如何在 Brightcove 信標中管理和建立即時伺服器端廣告設定。

    概覽

    Brightcove Beacon 支援即時活動中的伺服器端廣告插入 (SSAI)，因此可以在直播期間要求和顯示伺服器端廣告。

    該過程的高級視圖是：

    1. 創建一個廣告配置工作室。
    2. 為您的直播活動建立填色板。
    3. 在啟用 SSAI 的情況下建立即時活動。
    4. 將直播活動發佈到布萊特灣信標。
    本文件是對直播現場直播文檔，指導您將SSAI添加到現場活動中。有關如何創建活動的詳細信息，請參閱在Brightcove Beacon中創建現場活動。

    SSAI 在即時活動期間提供以下主要功能：

    • 由於廣告會拼接到伺服器端的即時影片串流，因此廣告不會受到廣告封鎖程式的影響。
    • 廣告會在編碼器傳送的提示點上插入，或者您可以使用即時模組控制室建立即時提示點。
    • 您可以配置填充石板來填補所有未使用的廣告時間。

    設定伺服器端廣告設定

    在這裡，您將創建一個 SSAI 廣告配置，用作您的廣告集將顯示在您的布萊特灣信標直播活動中。

    1. 登入工作室
    2. 在導覽標題中，按一下 [ 管理 ]，然後選取 [ 伺服器端廣告設定 ]。將會顯示所有廣告設定的清單。
    3. 若要建立新的廣告組態，請按一下 [ 建立廣告組態 ] 按鈕。
      伺服器端即時廣告設定也可以使用即時 API 來管理。如需詳細資訊，請參閱布萊特灣即時 API：伺服器端廣告插入 (SSAI) 。
    4. 從下拉菜單中選擇生活作為一種。
    5. 請確認您看到的是「伺服器端廣告設定」表單。
    6. 為您的廣告設定命名。
    7. 選取廣告伺服器回應：
      • 廣大
      • DFP (廣告規則)
      • 智慧型 XML
      請洽詢您的廣告營運管理員，以取得正確的值以選取。
    8. 輸入廣告代碼 (伺服器網址) 。
    9. 按一下「儲存」按鈕。

    配置填充板

    填色板是指即時直播廣告片段中出現空白，而無法填滿動態廣告時，會顯示的影片。如果要求的廣告短於廣告片段時間，則會顯示填色板，讓觀眾看不到空白播放器。

    請求伺服器端廣告之前，您應該至少設定一個填色板。

    若要新增填色板，請依照下列步驟執行：

    1. 點擊管理員 > 實時設置 Studio標頭中的鏈接。
      您必須是帳戶管理員才能存取 ADMIN 功能表。
    2. 按一下「 + 加入填滿板板」。你應該看到以下形式：
    3. 進入一個名稱為您的填充板。
    4. 輸入石板的「來源 URL 」。
    5. 按一下「儲存」。石板的攝取過程將開始。

    要刪除填充板，請點擊刪除按鈕（）旁邊的填充板岩。

    建立支援 SSAI 的直播活動

    為了使直播活動能夠使用SSAI（服務器端廣告插入），啟用服務器端廣告插入（SSAI）創建事件時必須選擇設置，並且應選擇先前創建的填充板進行播放。

    請依照下列步驟建立支援 Brightcove 信標中 SSAI 廣告的事件：

    1. 打開生活 Studio中的模塊。
    2. 從「管理事件」頁面中，按一下「建立事件」。
    3. 輸入您的活動詳細資料。有關使用“直播”模塊創建直播活動的完整信息，請參閱使用實時模塊創建和管理實時事件。
    4. 按一下「進階選項」以展開進階選項。
    5. 檢查一下啟用服務器端廣告插入（SSAI）。這將啟用伺服器端廣告插入功能。
    6. 選取「填滿板」。當直播廣告片段中有空白無法填滿動態播放的廣告時，即會顯示填色板。
    7. 勾選「廣告音訊」設定後，廣告上的音量會根據所設定的調整進行標準化。這可以用來協助確保廣告不會比內容大幅度大。布萊特灣建議您選取此選項。
    8. 請點擊建立活動
    9. 的控制室該頁面將打開，其中包含有關直播活動的詳細信息，包括流式傳輸端點（RTMP URL）和流名稱。這些值將由編碼器使用。
    10. 設定您的編碼器並啟動即時串流。
      若要在直播平台中開始串流，您需要編碼器，如需有關如何使用編碼器的資訊，請閱讀以下文件：

      逐步：使用即時模組與電流線上傳送

      逐步：使用開放式廣播軟體 (OBS) 的直播模組

    11. 默認情況下，控制室預覽播放器顯示無廣告流。廣告將不會顯示在控制室除非選擇了實時廣告配置。使用播放器下方的下拉式清單來選取廣告設定。
    12. （可選的）點擊加號按鈕（）以選擇其他廣告配置。這樣做會在控制室頁。

      按一下預覽播放器下方的音量按鈕，即可控制哪個播放器音訊串流啟動。

    要求廣告

    請按照以下步驟在直播中請求廣告控制室：

    1. 請點擊轉到廣告然後從下拉列表中選擇所需的廣告休息時間。
    2. 按一下 [ 插入]，即可在直播中插入廣告分享。如果廣告沒有填滿整個廣告突發時間，則會顯示填色板 (在建立活動時被選取)。

    廣告將會在控制室預覽播放器以及使用實時模塊發布的播放器中。此示例顯示了2個廣告在控制室。

    在布萊特灣燈塔發佈直播活動

    若要在 BrightBeacon 中使用伺服器端廣告發佈您的直播活動，請按照下列步驟進行：

    1. 在現場控制室點擊發布和嵌入在左側導航欄中。
    2. 選擇播放器配置。
    3. 在裡面實時播放器選項選擇一個可用的播放器。
    4. 選擇你的廣告配置先前在服務器端廣告設置。
    5. 選擇您的可用的播放器和廣告配置，將生成一個新的Player URL。如圖所示，複製URL的所選部分，從單詞開始生活。例： live.xxxxxxx
    6. 去布萊特灣信標平台
    7. 在Brightcove Beacon的上方菜單中，單擊廣告標籤。
    8. 點擊添加新配置按鈕。
    9. 檢查以確保您看到了新廣告形成。
    10. 給您的新廣告配置一個名稱。
    11. 在裡面實時廣告配置ID：，粘貼播放器網址ID您剛剛從Live Platform複製而來，例如： live.xxxxx。
    12. 來自提供者菜單選擇Brightcove SSAI。
    13. 點擊製作新廣告按鈕。
    14. 點擊上方菜單大事記標籤並創建或選擇您的直播活動。有關如何創建Brightcove Beacon直播活動的更多信息，請參見直播現場直播文件。
      為了使直播活動能夠在Brightcove Beacon中成功流式傳輸，工作編號和流視頻ID必須與當前直播事件的ID匹配。
    15. 在您的Brightcove Beacon直播活動中，單擊權利和計劃標籤。
    16. 在裡面權利類型下拉菜單，選擇廣告-AVOD。
    17. 在裡面廣告配置底部菜單中，選擇您先前創建的廣告配置。
    18. 點擊更新事件按鈕。
    19. 按一下畫面右上角的工具 (扳手) 按鈕。然後清除快取。
    20. 打開您的 Brightcove Beacon 應用程序並選擇您的活動（圖片上的活動名稱和圖片僅供例如用途）。
    21. 在您的Brightcove Beacon應用中，點擊玩按鈕。您現在應該可以看到您的直播活動串流，並由 SAI 插入來自 Video Cloud 的廣告。

    活動結束後，您可以將所有或部分內容儲存為影片雲端中的影片資產，或是可在 Brightcove Beacon 中顯示的可播放串流。直播活動的廣告不會被儲存，但會註冊傳送廣告的提示點，以便您可以在稍後錄製的直播活動的 VOD 中新增新廣告。

    有關如何創建剪輯的更多信息，請參見通過現場活動創建片段的部分直播現場直播文件。
