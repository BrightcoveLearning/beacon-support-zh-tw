    檢視分析資料

    在這個主題中，您將了解您的布萊特灣信標應用程式可以使用哪些分析資料，以及如何在 Google Analytics (分析) 中檢查該資料。
    布萊特灣信標不會將完整的分析傳送回視訊雲端。只有視訊啟動會回報至視訊雲端分析。

    簡介

    毫無疑問，您一定會想知道觀眾在您的布萊特灣信標應用程式中觀看的內容。要做到這一點，你將使用谷歌分析。布萊特灣將加載自定義的儀表板和自定義的谷歌數據工作室報告到您的谷歌分析帳戶。這兩種工具都能讓您深入了解客戶如何使用您的 Brightcove Beacon 應用程式。

    除了自訂的儀表板和報表之外，您還可以使用 Google Analytics (分析) 中的標準分析工具來檢視客戶行為。您可以使用谷歌收集的標準分析數據，另外 Brightcove 添加了其他實體使用，在本文件後面詳細說明。例如，電影在播放電影，默認的Google Analytics（分析）度量標準以及計算觀看時間（由Brightcove添加的度量標準）時調度事件。因此，在谷歌 Analytics 中，您可以通過觀眾播放電影的次數和總觀看時間看到「最多觀看」的電影。由 Brightcove 添加這些實體在前一段中提到的自定義儀表板和報告廣泛使用。

    檢視儀表板

    若要查看 Brightcove 安裝的 Google Analytics 儀表板，更重要的是儀表板顯示的數據，請按照下列步驟操作：

    1. 登錄谷歌分析。
    2. 點擊下拉菜單（屏幕截圖中以黃色突出顯示）以查看您的所有谷歌 Analytics 帳戶。
    3. 從三個面板的下拉式清單中，選取：
      1. 您的布萊特灣信標帳戶
      2. 您想要分析的應用程式
      3. 您的視圖（每個應用程序只有一個視圖）
    4. 在左側導航欄中，點擊客制化選項，然後打開儀表板以查看當前視圖的自定義儀表板。
    5. 如果儀表板沒有出現，您必須從清單中選取它：
    6. 您的儀表板會出現在畫面的右側。範例儀表板如下所示：

    檢視資料工作室報表

    根據谷歌：「Data Studio 是一個免費的工具，可將您的資料轉換為資訊豐富、易於閱讀、容易分享，以及完全可自訂的儀表板和報表。「

    要查看由布賴特灣安裝的自定義谷歌數據工作室報告，請按照下列步驟操作：

    1. 登錄到Google Data Studio（datastudio.google.com）。
    2. 您會看到登陸頁面上列出的報表清單 (以黃色反白顯示)：
    3. 按一下報告名稱以開啟它。
    4. 此處顯示範例報表的第一頁。報表左上角的注意是導覽 (以黃色反白顯示) 以查看其他頁面。

    檢視維度和量度

    如果您不熟悉分析維度和指標，這裡有一個非常簡單的解釋：
    • 維度是資料的屬性。例如，視圖來自的城市是一個維度。單一維度可以包含許多值，就像您擁有視圖的所有城市的名稱。
    • 指標是量化測量。因此，特定城市的會話數量將是一個指標。

    布萊特灣為您的分析報告添加了默認的谷歌量值的功能。您可以使用預設的 Google Analytics 維度和量度、分析報表的建構區塊，以及 Brightcove 新增的眾多維度和一個量度。這些額外的維度和一個量度會用於本文件前兩個區段所示的儀表板和 Data Studio 報表中。

    請注意，布賴特灣新增的尺寸和量度不同於每種類型的布萊特灣信標應用程式。例如，iOS 移動數據添加了 16 個自定義維度，而網站應用程序添加了 20 個自定義維度。若要在 Google Analytics (分析) UI 中查看維度和量度清單，請依照下列步驟執行：

    1. 登錄谷歌分析。
    2. 點擊下拉菜單（屏幕截圖中以黃色突出顯示）以查看您的所有谷歌 Analytics 帳戶。
    3. 從三個面板的下拉式清單中，選取：
      1. 您的布萊特灣信標帳戶
      2. 您想要分析的應用程式
      3. 您的視圖（每個應用程序只有一個視圖）
    4. 在左側導航欄中，點擊管理員選項。
    5. 請點擊自定義定義。
    6. 當您單擊自定義定義您將看到以下鏈接自定義尺寸和自定義指標。
    7. 點擊自定義尺寸鏈接以查看此應用的尺寸。尺寸會隨應用程式而變更。
    8. 點擊自定義指標鏈接以查看單個主題，觀看時間。這是布萊特灣信標應用程式的唯一自訂度量。

    報告範例

    本文件是為了讓你知道你可以從谷歌分析使用的信息。本文件並非意圖成為如何使用 Google Analytics 與您的布萊特灣信標應用程序的完整教學指南。話雖如此，文件的下一節將詳細說明如何查看一些關於您的 Brightcove Beacon 應用程序的常見報告。

    觀眾資訊

    您可以依照下列步驟檢視對象的分析：

    1. 按照上述任何一個區段的前三個步驟，選擇您的應用程式/檢視。
    2. 在左側導航中，首先單擊聽眾然後總覽。

    3. 在屏幕右側，您可以使用不同的指標（例如，語言，應用程式資訊和設備。您可以按一下特定量度以取得更多詳細資訊。

    螢幕檢視資訊

    當然，您會想知道觀眾在您的應用程式中執行的動作。為此的選項是行為。請按照以下步驟查看屏幕瀏覽信息：

    1. 在左側導航欄中，點擊行為→網站內容→所有頁面。

    2. 在畫面右側，您會看到最常檢視的畫面清單。

    播放和觀看時間資訊

    仍在行為選項，您可以查看視頻資產點擊播放按鈕的次數，以及實際觀看該資產的時間。請依照下列步驟取得該資訊：

    1. 雖然仍在行為選項，單擊活動→熱門活動。
    2. 在裡面活動類別列表中，單擊播放電影
    3. 在裡面活動類別列表中，單擊電影。
    4. 在裡面均勻動作列表中，單擊玩。
    5. 請注意，清單會顯示每個資產按一下播放按鈕的次數。
    6. 請注意，您可以透過將適當的資料行新增至表格，以查看一或多個自訂維度的資訊。為此，請點擊次級尺寸下拉菜單中，您可以在其中看到自定義尺寸組。
    7. 您現在可以打開自定義尺寸分組並選擇一個尺寸。
    8. 查看示例表類型自定義維度已添加。
    9. 請注意事件動作您還可以看到列表觀看時間，當然會使用您的自定義指標觀看時間。

    谷歌分析學習資源

    您剛剛刮傷了谷歌分析在本文檔中的表面。您可以將所選資產的每日或每週摘要資訊放在一起的自訂報告。一個開始學習更多有關Google Analytics（分析）的旅程的好地方可能是Google Analytics Academy和/或Google Analytics（分析）文檔。
