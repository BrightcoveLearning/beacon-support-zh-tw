介紹

客戶端信息從應用程序傳遞到SSAI，然後SSAI將此數據包含在客戶的VAST代碼中，然後VAST代碼可讓客戶根據此數據來定位廣告。這樣做的目的是增加客戶廣告庫存的價值，從而增加他們業務的廣告收入。

您可以設置如何將此數據傳遞給他們的廣告服務器或廣告提供商。

好處

改善廣告定位

  • 用戶會話 ID：當用戶打開應用程序時，我們會在每台設備上生成一個會話 ID。這允許您為該會話設置頻次上限，排除競爭性廣告一起投放，並在該會話中按順序輪播廣告。這些定位功能對於直銷廣告和程序化廣告都是必要的
  • 程序化廣告的更多數據：移動應用程序和CTV應用程序不支持Cookie，這是網絡上基於用戶/設備的廣告定位的主要方法。通過添加諸如設備ID之類的參數，您可以將這些有價值的信息傳遞給程序化廣告購買者，這將增強他們的廣告定位功能，從而帶來更高的每千次展示費用。

支持廣告庫存拆分

對於在Amazon，Roku和Samsung（在美國）上運行的應用，您需要使用該平台按70/30比例分配廣告資源/收入。信標客戶通過70％的廣告資源獲利，而平台則獲得了剩餘30％的廣告資源。通過這種增強的廣告定位，您可以在他們的所有應用中使用一個廣告標籤，並將設備參數傳遞給他們的廣告服務器。他們將在廣告服務器內部實施業務邏輯，以按設備劃分廣告資源。平台還需要傳遞 LAT - 有限的廣告定位 - 用於廣告庫存拆分，這是我們現在發送到您的廣告服務器的參數。

用戶隱私改進

  • LAT“受限廣告定位"-允許Beacon應用程序的用戶根據用戶行為選擇退出廣告定位。
  • 用戶可以隨時重置我們分配給設備的唯一設備 ID。

筆記

  • 此功能適用於所有平台/設備 iOS、Apple TV、Android、Roku、Fire TV、Web 和智能電視。
  • 信標應用廣告定位可以與VOD SSAI和Live SSAI一起使用。

點播步驟

視頻雲部分 - VOD

  1. 通過創建廣告配置Studio中的服務器端廣告設置頁：
    1. 點擊創建廣告配置然後選擇視頻點播
    2. 為廣告配置添加名稱<./ li>
    3. 選擇您的廣告響應 - 建議使用 VMAP 或 Vast。如果您使用Vast，則需要在視頻中添加提示點，以指定應在何處顯示廣告插播廣告。
    4. 粘貼到您的廣告代碼中（來自廣告提供商）。
    5. 複製您的廣告配置ID，因為您需要燈塔廣告配置。

信標部分-視頻點播

  1. 打開Beacon Classic用戶界面，然後轉到“廣告"標籤：
    1. 點擊添加新配置
    2. 命名廣告配置
    3. 選擇Brightcove SSAI
    4. 粘貼在SSAI廣告配置ID進入VOD廣告配置ID
  2. 在Studio和Beacon Classic中設置廣告配置後，您需要實施客戶端參數，這些參數將傳遞給客戶廣告服務器。客戶端參數可以在Playback API（在VMAP /清單輸入點中提供）中傳遞-這些參數將以{{url.xxx}}宏，其中xxx對應於在回放API URL中提交的參數密鑰。
    1. 在廣告定位信標中的字段。輸入將是查詢形式的字符串。例子： d= {PLATFORM} &os= {OS} &man= {MFGR} &mo= {MODEL} &appid= {APP_ID} &lan= {LOCALE} &確實= {RDID} &dt= {IDTYPE} &lat= {IS_LAT} &sid= {SID}
    2. 您可以在等號前配置鍵 .他們必須匹配添加到 SSAI 廣告代碼的鍵。
    3. 接下來，在SSAI Studio設置頁。以下是在 Beacon Classic Ad-Targeting 中輸入並添加到 SSAI 廣告標籤的參數示例： d= {PLATFORM} &os= {OS} &appid= {APP_ID} &確實= {RDID} &dt= {IDTYPE} &lat= {IS_LAT} &sid= {SID}
    4. 這些參數在這個 Ad 標籤中被翻譯成這樣： https://ads.brightcove.com/ads?tech=dfpadrules&dur=15&pre=3&d= {{url.d}} &os= {{url.os}} &appid= {{url.appid}} &確實= {{url.did}} &dt= {{url.dt}} &lat= {{url.latl}} &sid= {{url.sid}}

現場步驟

  1. 首先，構造廣告定位參數字符串，因為在SSAI和Beacon配置中都將需要它。您可以為 VOD 重複使用參數，或選擇不同的集合。

    例子：

    ip= {{client.ipaddress}} &num= {{random.int32}} &d= {{PLATFORM}} &os= {{OS}} &確實= {{RDID}} &dt= {{IDTYPE}} &lat= {{IS_LAT}}
  2. 從以下位置創建廣告配置Studio中的“服務器端廣告設置"頁面：
    1. 點擊創建廣告配置然後選擇居住
    2. 為廣告配置添加名稱
    3. 選擇廣闊的您的廣告響應
    4. 粘貼到您的廣告代碼中，附加您的廣告定位參數查詢字符串。

      例子：

      https://ad-provider-host.com/path/to/ad-handler?ip= {{client.ipaddress}} &num= {{random.int32}} &d= {{PLATFORM}} &os= {{OS}} &確實= {{RDID}} &dt= {{IDTYPE}} &lat= {{IS_LAT}}
    5. 複製信標廣告配置的廣告配置ID
  3. 打開Beacon Classic用戶界面，然後轉到“廣告"標籤
    1. 點擊添加新配置
    2. 命名廣告配置
    3. 選擇Brightcove SSAI
    4. 將SSAI廣告配置ID粘貼到實時廣告配置ID
  4. 將您的廣告定位參數字符串粘貼到廣告定位場地。

    例子：

    ip= {{client.ipaddress}} &num= {{random.int32}} &d= {{PLATFORM}} &os= {{OS}} &確實= {{RDID}} &dt= {{IDTYPE}} &lat= {{IS_LAT}}
