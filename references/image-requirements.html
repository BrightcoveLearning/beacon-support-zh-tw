    圖片及其規格

    在本主題中，您將了解布萊特灣信標體驗所需的影像。還詳細說明了所需的圖像尺寸，縱橫比等。攝錄到視頻雲的視頻需要一些圖像，而有些則用於布萊特灣信標。

    視頻雲圖像

    每當將視頻攝取到Video Cloud中時，都會自動創建兩個圖像並將其顯示在視頻的屬性中。創建的兩個圖像是海報和一個縮圖。圖像顯示在圖片 Studio中的部分。您將使用這些上傳圖像在Brightcove Beacon中使用，但必須更改默認的攝取配置文件才能正常使用。以下警報框中顯示的步驟顯示瞭如何進行更改。如果不進行更改，則正在上傳的肖像圖像將保持縮略圖大小，並且在Brightcove Beacon應用中看起來很差。

    您必須更改默認的攝取配置文件。請依照下列步驟執行：
    1. 在Studio中轉到管理→提取配置文件。
    2. 單擊您當前默認配置文件的標題，該標題標有上載預設標籤。
    3. 按一下「複製」按鈕。
    4. 找到哪裡圖片被定義。確切的JSON可能看起來與下面顯示的略有不同，具體取決於您複製的配置文件。
    5. 更改海報和縮略圖的寬度和高度，如下所示：。

      舊設定

      <類別 = "語言-JSON」> {
  「媒體類型」：「圖像」，
  「格式」:「JPG」,
  「標籤」：「海報」，
  「寬度」：1280
  「高度」：720，
  「ID」：「五十字架」
}，
{
  「媒體類型」：「圖像」，
  「格式」:「JPG」,
  「標籤」:「縮圖」,
  「寬度」：160
  「高度」：90，
  「ID」：「五千八百年」
 }

      新設定

      <類別 = "語言-JSON」> {
  「媒體類型」：「圖像」，
  「格式」:「JPG」,
  「標籤」：「海報」，
  「寬度」：4000，
  「高度」：2250，
  「ID」：「五十字架」
}，
{
    「媒體類型」：「圖像」，
    「格式」:「JPG」,
    「標籤」:「縮圖」,
    「寬度」：3000，
    「高度」：2000年
    「ID」：「五千八百年」
 }
    6. 按一下「儲存」。
    7. 然後點擊將此設為我的默認個人資料鏈接。
    8. 檢查以確保配置文件現在被標記為您的上傳默認設置。
    現在，您可以使用Video Cloud Studio為Brightcove Beacon上載圖像了。

    使用Studio界面上傳兩個圖像。您將決定代表您的視頻的圖像，並採取符合上述要求的適當屏幕截圖。然後，您將兩個圖像上傳到Video Cloud。

    的海報和一個縮圖是Brightcove Beacon中使用的圖像的佔位符。不要遵循Video Cloud Studio中顯示的圖像建議！相反，請使用以下準則：

    海報 縮圖
    橫向影像 (16:9)：
    最小值 寬度：1000 像素 高度：563 像素
    最大值 寬度：4000 像素 高度：2250 像素
    		海報影像 (2:3)
    最小值 寬度：1000 像素 高度：1500 像素
    最大值 寬度：2000 像素 高度：3000 像素

    屏幕截圖左側的圖像是肖像圖像的示例用法。如果您看到非常像素化的圖像，則將知道需要更新默認的攝取配置文件。

    公司標誌

    您將需要提供公司徽標以用於Brightcove燈塔體驗。請遵循以下準則處理圖像：

    規範 價格
    格式 .png
    長寬比 1：1
    大小 1500px x 1500px

    在Brightcove Beacon中工作

    在Brightcove Beacon中輸入信息時，系統會要求您輸入非文字數據。這意味著您正在創建的資產的圖像。這些圖像可以分為三組，分別是：

    • 系列/季節
    • 現場活動
    • 頻道徽標

    組圖像的詳細信息如下。

    系列/季節

    對於系列/季節，頁面將類似於以下內容（沒有紅色可選的文字在縮圖，您無需提供圖片）：

    遇到此屏幕時，請提供一個JPEG格式的圖片人像海報，風景海報和寬幅橫幅符合以下要求。您無需提供以下圖片：縮圖。如果未提供，則會從「人像海報」產生縮圖影像。

    影像 需求
    人像海報 海報影像 (2:3)
    最小值 寬度：1000 像素 高度：1500 像素
    最大值 寬度：2000 像素 高度：3000 像素
    風景海報 橫向影像 (16:9)：
    最小值 寬度：1000 像素 高度：563 像素
    最大值 寬度：4000 像素 高度：2250 像素
    寬橫幅 風景圖片（21：9）：
    最小值 寬度：1000 像素 高度：428 像素
    最大值 寬度：4000 像素 高度：1714 像素

    現場活動

    對於現場活動，頁面將類似於以下內容（沒有紅色可選的文字在縮圖和寬幅橫幅，您無需提供圖像）：

    遇到此屏幕時，請提供一個JPEG格式的圖片人像海報和風景海報符合以下要求。您不需要為「縮圖」和「寬廣」橫幅提供影像。如果未提供，則縮略圖圖像將從肖像海報生成，而寬幅橫幅則由風景海報生成。

    影像 需求
    人像海報 海報影像 (2:3)
    最小值 寬度：1000 像素 高度：1500 像素
    最大值 寬度：2000 像素 高度：3000 像素
    風景海報 橫向影像 (16:9)：
    最小值 寬度：1000 像素 高度：563 像素
    最大值 寬度：4000 像素 高度：2250 像素

    頻道徽標

    頻道徽標圖像由縮略圖背景和圖像（具有或不具有透明性）組成，兩者均具有16：9的縱橫比。下圖顯示瞭如何創建頻道徽標：

    頻道徽標圖片 橫向影像 (16:9)：
    所需尺寸 寬度：1000 像素 高度：563 像素
