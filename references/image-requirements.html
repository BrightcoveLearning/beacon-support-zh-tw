視頻雲圖片

每當將視頻攝取到 Video Cloud 中時，都會自動創建兩個圖像並顯示在視頻的屬性中。創建的兩個圖像是一個 海報 和一個 縮略圖.圖像顯示在圖片工作室中的部分。您將使用這些上傳圖像在 Brightcove Beacon 中使用，但必須更改默認攝取配置文件以獲得正確的功能。以下警報框中顯示的步驟向您展示瞭如何進行更改。如果您不進行更改，則正在上傳的肖像圖像將保持縮略圖大小，並且在Brightcove Beacon應用中看起來很差。

您必須更改默認的攝取配置文件。按著這些次序：
  1. 在Studio中轉到管理→提取配置文件。
  2. 單擊您當前默認配置文件的標題，該標題標有上載預設標籤。
  3. 點擊複製按鈕。
  4. 找到哪裡圖片被定義。確切的 JSON 可能與下面顯示的略有不同，具體取決於您複製的配置文件。
  5. 更改海報和縮略圖的寬度和高度，如下所示：。

    舊設定

    {
  "media_type": "image",
  "format": "jpg",
  "label": "poster",
  "width": 1280,
  "height": 720,
  "id": "559a8b0ee4b07cbb40be7d22"
},
{
  "media_type": "image",
  "format": "jpg",
  "label": "thumbnail",
  "width": 160,
  "height": 90,
  "id": "559a8b0ee4b07cbb40be7d23"
}

    新設定

    {
  "media_type": "image",
  "format": "jpg",
  "label": "poster",
  "width": 4000,
  "height": 2250,
  "id": "559a8b0ee4b07cbb40be7d22"
},
{
    "media_type": "image",
    "format": "jpg",
    "label": "thumbnail",
    "width": 3000,
    "height": 2000,
    "id": "559a8b0ee4b07cbb40be7d23"
}
  6. 點擊保存。
  7. 然後點擊將此設為我的默認配置文件關聯。
  8. 檢查以確保配置文件現在被標記為您的默認上傳文件。
您現在可以使用 Video Cloud Studio 為 Brightcove Beacon 上傳圖像。

使用 Studio 界面上傳兩張圖片。您將決定代表您的視頻的圖像，並拍攝滿足上述要求的適當屏幕截圖。然後，您將這兩個圖像上傳到 Video Cloud。

這 海報 和一個 縮略圖 是 Brightcove Beacon 中使用的圖像的佔位符。 請勿遵循Video Cloud Studio中顯示的圖像建議！ 相反，請使用以下準則：

海報 縮略圖
風景圖片（16：9）：
最低限度 寬度：1000像素 高度：563px
最大 寬度：4000像素 高度：2250像素
海報圖片（2：3）
最低限度 寬度：1000像素 高度：1500px
最大 寬度：2000像素 高度：3000像素

屏幕截圖左側的圖像是肖像圖像的示例使用。如果您看到非常像素化的圖像，則將知道需要更新默認的攝取配置文件。

公司標誌

在 Brightcove Beacon 工作

在 Brightcove Beacon 中輸入信息時，系統會要求您提供非文本數據 .這意味著您當前正在創建的資產的圖像。這些圖像可以分為三組，分別是：

  • 系列/季節
  • 現場活動
  • 頻道徽標

組圖像的詳細信息如下。

系列/季節

對於系列/季節，頁面將類似於以下內容（沒有紅色可選的文字在縮圖，您無需為其提供圖片）：

當您遇到此屏幕時，請提供JPEG格式圖像為人像海報 , 風景海報和寬幅橫幅符合以下要求。您不需要提供圖像縮略圖 .如果未提供，則將根據人像海報生成縮略圖圖像。

圖像 要求
人像海報 海報圖片（2：3）
最低限度 寬度：1000像素 高度：1500px
最大 寬度：2000像素 高度：3000像素
風景海報 風景圖片（16：9）：
最低限度 寬度：1000像素 高度：563px
最大 寬度：4000像素 高度：2250像素
寬幅橫幅 風景圖片（21：9）：
最低限度 寬度：1000像素 高度：428像素
最大 寬度：4000像素 高度：1714像素

現場活動

對於現場活動，頁面將顯示類似於以下內容（沒有紅色可選的文字在縮圖和寬幅橫幅，您無需為其提供圖像）：

當您遇到此屏幕時，請提供JPEG格式圖像為人像海報和風景海報符合以下要求。你不需要提供圖像縮略圖和寬幅橫幅 .如果未提供，則縮略圖圖像將從肖像海報生成，而寬幅橫幅則由風景海報生成。

圖像 要求
人像海報 海報圖片（2：3）
最低限度 寬度：1000像素 高度：1500px
最大 寬度：2000像素 高度：3000像素
風景海報 風景圖片（16：9）：
最低限度 寬度：1000像素 高度：563px
最大 寬度：4000像素 高度：2250像素
