    一般組態設定

    在本主題中，您將瞭解在 Brightcove Beacon 中建立或修改應用程式時所使用的常用組態設定。

    概覽

    本主題是與您的 Brightcove Beacon 應用程式相關聯的常見組態設定彙編。您可以在 Brightcove Beacon 中完成特定工作的其他文件中找到這些設定，包括下列標籤區段：

    • 配置
    • 頁面
    • 頻道
    • 電影
    • 播放清單
    • 活動/系列
    • 活動
    • 商務/包裝/串流/進階

    權利與排程

    的權利和計劃部分可讓您定義播放權限並設置應用的發佈時間表。

    以下是新增或編輯頁面時的範例：

    該表包含您將在權利和計劃標籤：

    欄位 必填 描述
    狀態 是 選取下列其中一種類型：
    • 已發佈
    • 未發佈
    權限類型 是 請指定您的頻道是否支援下列值的廣告：
    • 免費
    • 廣告-AVOD (廣告視頻點播)
    • 廣告-SVD (訂閱視頻點播)
    • 繼承-這意味著權利將從本賽季所屬的系列繼承。
    權限開始日期 是 設定應用程式的開始日期和時間。（也，開始時間）
    權限結束日期 是 設定應用程式的結束日期和時間。（也，時間結束）
    要發佈的裝置 是 新增您想要發佈的裝置。選取 [ 選取/取消選取所有裝置 ] 以查看可用裝置的清單。
    允許的位置 是

    選取您要允許使用者檢視內容的地理位置。在內部單擊允許的位置框以查看位置列表。如果沒有限制，請輸入世界。

    如果您沒有指定至少 1 個允許的位置，則您可能會從您的應用程序看到以下消息：
    拒絕的位置 否 選取您不想允許使用者檢視內容的地理位置。在內部單擊拒絕地點框以查看位置列表。
    最大裝置 否 設定帳戶的使用中連線裝置數目上限。整數值。
    最大串流 否 設定可從同一帳戶觀看的串流數目上限。整數值。

    設定例最大流至4和最大設備至6：這意味著 4 人可以串流（流）視頻，但你可以有 6 個人連接（設備）。

    文字資料

    的文字數據部分可讓您添加可顯示的文本。這是顯示在應用程式中的名稱。

    以下是新增或編輯頁面時的範例：

    該表包含您將在文字數據標籤：

    欄位 必填 注意事項
    標題 是 顯示的應用程式標題。你會發現你已經定義的每個語言的輸入字段。

    通常會在基本數據選項卡名稱。此名稱僅供內部參考，而文字數據標籤的標題是用戶將在應用程序中看到的內容。

    非文字資料

    的非文字數據部分中，您可以上傳應用的背景圖像和圖標圖像。

    以下是新增或編輯頁面時的範例：

    該表包含您將在非文字數據標籤。請注意，您可以為您定義的每種語言上傳圖片。

    欄位 必填 注意事項
    圖示 是 您想要表示您的頁面的圖像。
    暫留圖示 是 當使用者將游標停留在圖示影像上時，會顯示此影像。
    水平背景 是 當使用者裝置處於橫向模式時，會顯示背景影像。
    垂直背景 是 當使用者裝置處於縱向模式時，會顯示背景影像。

    允許的上傳大小上限為 500 KB。
    有關在Brightcove Beacon中上傳圖像的詳細信息，請參閱圖像及其規格文件。
