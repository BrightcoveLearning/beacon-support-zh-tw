任務總結

以下是您作為 Brightcove Beacon 客戶應該/必須在 Video Cloud Studio 中執行的任務列表：

  • 添加視頻資產
  • 創建自定義字段
  • 添加與視頻資產關聯的圖像
  • 添加權限管理信息
  • 將系列/季的劇集數據添加到視頻資產
  • 創建視頻播放列表
  • 創建Brightcove Live活動
  • 創建SSAI廣告配置

每個任務將在本文檔的後面部分通過文檔的鏈接進行進一步說明，其中提供了有關如何執行特定任務的說明。

本文檔不是每項任務的“操作方法”手冊，但每個部分都會將您鏈接到包含有關如何完成任務的步驟的相應文檔。

添加視頻資產

Video Cloud 是用於上傳您在 Brightcove Beacon 中使用的視頻的工具。在本文檔中，您將看到 Video Cloud Studio 是使用 Brightcove Beacon 應用程序時常用的工具。

以下是將您的視頻導入 Video Cloud 的資源：

  • 視頻雲基礎：上傳視頻 - 一個 1 分 30 秒的視頻和四個步驟，用於快速開始將內容上傳到 Video Cloud。
  • 使用上傳模塊上傳視頻 - 關於使用 Video Cloud 上傳視頻的完整文檔。
  • 管理影片 -涵蓋整個視頻管理工作流程的登錄頁面，包括有關上傳視頻，播放列表，組織帶有文件夾的視頻等的文檔。

創建自定義字段

Video Cloud 允許您使用自定義字段為您的視頻添加元數據。擁有視頻後，自定義字段將在準備要在Brightcove Beacon中使用的視頻中扮演重要角色。您添加到與 Brightcove Beacon 一起使用的 Video Cloud 視頻的自定義字段分為三類。這些類別以及每個類別的簡要說明是：

  • 一般的 -有關視頻內容的信息，包括：
    • 類型
    • 簡短說明和詳細說明
    • 生產年份
    • 投
    • 作家
    • 等等。
  • 權限管理 - 支持貨幣化和訪問限制。例如，廣告類型和視頻可用日期。
  • 劇集數據 -如果視頻是系列的一集，則可以定義系列，季和集號。

這創建視頻云自定義字段文檔專門用於幫助您創建所需的自定義字段。

在 Video Cloud Studio 中將值分配給這些自定義字段。

添加權限管理信息

如前所述，自定義字段用於添加用於控制獲利和訪問限制的權限管理信息。例如，字段

  • 信標。權利。指數。開始日期
  • 信標。權利。指數。結束日期

定義何時可以訪問特定視頻資產。

此外，以下文檔顯示瞭如何通過廣告和訂閱來實現獲利：

  • 實施VOD SSAI ;看到設置自定義字段值文檔部分，用於向相應的自定義字段提供值。
  • 使用訂閱包

權限管理自定義字段在 Video Cloud Studio 中分配值。

如果特定視頻是系列的一集，則使用自定義字段來分配將視頻與正確的系列，季和集號相關聯的集數據。此示例顯示使用的自定義字段以及每個字段的特定值。

創建視頻播放列表

在決定要在應用程序頁面上顯示哪些視頻內容時，Brightcove Beacon中會充分利用播放列表。您在 Video Cloud Studio 中創建播放列表。

以下是將您的視頻導入 Video Cloud 的資源：

  • 視頻雲基礎：創建手動播放列表
  • 視頻雲基礎：創建智能播放列表
  • 創建和管理播放列表
在 Video Cloud Studio 中創建的播放列表只能包含視頻。如果您希望創建包含系列、現場活動等的播放列表，則必須在 Brightcove Beacon 中創建這些播放列表。

創建Brightcove Live活動

如果您想在Brightcove Beacon應用程序中播放實時流事件，則必須在Video Cloud Studio中完成部分過程。這直播活動文檔將引導您完成整個過程。

創建SSAI廣告配置

服務器端廣告插入 (SSAI) 可用於 Brightcove Beacon。要實現這一點，您必須在 Video Cloud Studio 中創建廣告配置。這實施 VOD SSAI文檔包括如何創建該廣告配置，以及為 Brightcove Beacon 設置 VOD SSAI 的端到端流程。
