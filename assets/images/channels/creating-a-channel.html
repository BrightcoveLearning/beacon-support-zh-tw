描述

創建頻道以流式傳輸24/7直播事件，就像常規電視頻道一樣。頻道的內容可以通過一組 VOD 生成，只要頻道可用，這些 VOD 就會廣播，也可以通過直接傳輸直播事件來生成。有關如何創建直播活動的更多信息，請訪問Brightcove信標實時文檔。

除了經典的信標頻道，您現在還可以使用雲播出頻道為您的頻道提供內容。Cloud Playout 頻道的主要優勢是 Brightcove 生成 EPG - 您不需要第三方服務。

基本數據部分

要在 Brightcove Beacon 中創建頻道，請按照下列步驟操作：

  1. 登錄 Beacon Classic。
  2. 在頂部導航中，選擇頻道數。
  3. 點擊新增頻道按鈕。
  4. 在裡面基本數據標籤，定義一個名稱為您的頻道。
  5. 如果您將Cloud Playout頻道用作信標頻道的來源，請輸入Cloud Playout頻道ID：

    要查找 Cloud Playout 頻道 ID，請參閱獲取 Beacon 頻道的信息 .

  6. 在裡面內容分級下拉菜單中，根據您的頻道流式播放國家/地區的內容分級系統選擇頻道的內容分級。在示例圖片中，它是針對美國的電視台。
  7. 在裡面類型詳情部分中，單擊體裁下拉菜單為您的頻道選擇類型。（例子：恐怖，戲劇等。）
  8. 如果找不到適合您頻道的流派，則可以通過點擊創建體裁按鈕。
    9. 這子流派下拉菜單，此時不起作用。

文本數據

這文本數據部分允許您為頻道添加可顯示的文本。

  1. 選擇文本數據標籤。
  2. 在裡面標題字段，為您的頻道定義標題。
  3. 在裡面描述字段，為您的頻道定義說明。
  4. 在裡面概要字段，為您的頻道定義摘要。
    5. 您可以添加一個文本數據您將在頻道中使用的每種不同語言的部分。

非文字數據

這非文本數據部分允許您為頻道添加縱向和橫向圖像。您將提供縱向和橫向圖像。確保圖像滿足以下要求：
圖像 要求
人像海報 海報圖片（2：3）
最低限度 寬度：1000像素 高度：563px
最大 寬度：4000像素 高度：2250像素
風景海報 風景圖片（16：9）：
最低限度 寬度：1000像素 高度：563px
最大 寬度：4000像素 高度：2250像素

  1. 選擇非文本數據標籤。
  2. 添加一個人像海報通過在框部分中拖放圖片來獲取您頻道的圖片。
  3. 添加一個風景海報通過在框部分中拖放圖片來獲取您頻道的圖片。
只需選擇所需的語言選項卡，您就可以為頻道中使用的每種不同語言添加不同的縱向和橫向圖像集。

標籤

這標籤選項卡此時不起作用。

元數據

這元數據選項卡此時不起作用。

流

這溪流部分允許您設置播放 URL 的選項和權限。

  1. 選擇流標籤。
  2. 選擇您的提供者 .您很可能只有一個提供商，該提供商引用您的 Brightcove Video Cloud 帳戶。
  3. 點擊添加新的流按鈕。
  4. 在裡面流視頻ID字段中，粘貼您的視頻雲直播活動中的視頻ID。如果您將Cloud Playout頻道用作信標頻道的來源，請參見獲取有關信標頻道的信息。
  5. 在裡面要發布的設備部分中，添加您要發佈到的設備。選擇選擇/取消選擇所有設備查看可用設備列表。
  6. 在裡面流類型選擇居住。
  7. 請注意DRM類型和流格式值不需要更改。

權限和調度

這權限和調度部分是您設置發布狀態和可用性的地方。

  1. 選擇權限和調度標籤。

  2. 確保地位設定為已發表為您的頻道。
  3. 為了可用性詳細信息，請以以下內容為指導來完成信息：
    場地 必要 筆記
    權限類型 是的 根據您的獲利模式選擇：
    • 自由
    • 廣告-AVOD：對於通過廣告獲利的渠道。
    • 訂閱 - 點播：對於通過訂閱獲利的頻道。
    開始時間 是的 為您的頻道安排一個開始時間。
    時間結束 是的 為您的頻道安排一個結束時間。
    最大流 不適用 目前無法運作。
    要發布的設備 是的 添加您要發佈到的設備。選擇選擇/取消選擇所有設備查看可用設備列表。
    允許的位置 是的 選擇要流式傳輸內容的位置。
    拒絕地點 沒有 選擇您不想流式傳輸內容的位置。

    要添加或編輯地理位置，請導航至設置進而地點 .
  4. 請務必點擊建立新頻道按鈕保存您的工作。

將頻道添加到電子節目指南（EPG）

創建頻道後，最後一步是將其添加到Brightcove Beacon EPG時間表中。有關如何執行此操作的信息，請訪問創建電子節目指南（EPG）文檔。

請注意，如果您使用 Cloud Playout 頻道作為 Beacon 頻道的來源，Brightcove 將為您創建 EPG，您無需採取進一步行動。
